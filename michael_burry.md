# Michael Burry 投資框架

## QUICK_FACTS
- **基金**：Scion Asset Management（2013-至今）
- **規模**：~$250M
- **核心風格**：逆向深挖 + 極端集中 + 獨立研究
- **最佳適用**：泡沫識別、逆向機會、做空策略

> **風格**：逆向深挖、集中持倉、不從眾

## 核心原則
1. **獨立研究**：不依賴賣方報告；自己挖 SEC 文件、數據、細節。
2. **極端逆向**：共識錯誤時下重注；接受長期被市場「打臉」。
3. **集中持倉**：找到高確信機會就重倉；分散是「無知的保護」。
4. **價值錨定**：用清算價值、重置成本等硬指標估值。
5. **長期視野**：願意等待數年讓論點兌現。

## 經典戰役
| 年份 | 事件 | 邏輯 |
|------|------|------|
| 2005-2008 | 做空次貸（CDS） | 次級房貸違約率必然飆升 |
| 2019 | 做多 GameStop | 被過度做空、有資產價值 |
| 2021 | 警告通膨/泡沫 | 流動性過剩、估值極端 |
| 2022-23 | 做空 TSLA/AAPL | 估值過高、周期風險 |

## 研究方法
1. **讀原始文件**：10-K、10-Q、信用評級報告、貸款池數據。
2. **找不一致**：管理層說法 vs 財務數據 vs 行業趨勢。
3. **計算硬底**：清算價值、有形資產、現金流折現。
4. **測試共識**：為什麼市場這樣定價？共識可能錯在哪？

## 快速檢查清單
- **共識是什麼？** 市場為何這樣定價？
- **共識哪裡錯？** 有什麼數據/邏輯被忽略？
- **硬底在哪？** 最壞情況的清算/殘值？
- **時間框架？** 論點兌現需要多久？能撐住嗎？
- **倉位規模？** 確信度與倉位匹配嗎？

## 與宏觀/週期的連結
- **信用泡沫末期**：尋找被忽視的結構性風險（如 2005 次貸）。
- **極端估值**：泡沫頂部做空高估值成長股。
- **通膨/利率轉向**：提前佈局受益於利率上升的標的。
- **流動性收緊**：高槓桿、負現金流公司最危險。

## DECISION_RULES（決策規則）

### 逆向買入規則
```
IF 共識明顯錯誤 AND 有數據/邏輯支持
   THEN 深入研究
   BECAUSE 共識錯誤是超額回報來源

IF 股價像「路殺」(road kill) AND 資產價值仍在
   THEN 考慮買入
   BECAUSE 極端低估

IF 清算價值 > 市值 AND 有催化劑
   THEN 考慮買入
   BECAUSE 硬底保護

IF 高確信 AND 論點獨立驗證
   THEN 重倉
   BECAUSE 集中在最好的機會
```

### 做空規則
```
IF 估值極端 AND 需要「信仰」合理化
   THEN 考慮做空
   BECAUSE 泡沫終會破

IF 結構性風險被市場忽視（如 2005 次貸）
   THEN 深入研究做空機會
   BECAUSE 共識盲點

IF 流動性收緊 AND 高槓桿/負現金流公司
   THEN 做空候選
   BECAUSE 最危險的標的
```

### 研究規則
```
IF 賣方報告說好
   THEN 獨立驗證
   BECAUSE 不依賴賣方

IF 管理層說法
   THEN 對照財務數據與行業趨勢
   BECAUSE 找不一致

IF 市場定價
   THEN 問「為什麼？共識哪裡錯？」
   BECAUSE 測試共識
```

### 風控規則
```
IF 論點兌現需要數年
   THEN 確保資金續航
   BECAUSE 可能長期被「打臉」

IF 做空
   THEN 嚴格控制倉位
   BECAUSE 無限虧損風險

IF 時機太早
   THEN 可以承受但要有預案
   BECAUSE 時機是最難的
```
