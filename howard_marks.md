# Howard Marks 投资框架

## QUICK_FACTS
- **基金**：Oaktree Capital
- **规模**：$190B
- **核心风格**：周期意识 + 第二层思考 + 风险控制

> 重点：周期、风险控制、第二层思考；「当下处于周期的哪里？」是永恒问题。

## 核心原则
1. **周期意识**：经济、信用、情绪皆呈周期；极端后必回归。
2. **第二层思考**：共识已被定价，超额收益来自「共识是错的」或「定价不足」。
3. **风险 ≠ 波动**：真正风险是永久亏损；波动只是价格噪音。
4. **非对称回报**：下档保护优先，有限损失无限上涨。
5. **逆向 + 纪律**：群众恐慌时买、疯狂时卖，但需有分析支持。

## 周期定位框架
| 阶段 | 特征 | 行动 |
|------|------|------|
| 底部 | 信用冻结、恐慌抛售、估值极低 | 积极进场；强调存续力 |
| 复苏 | 利差收窄、风险偏好回升 | 持有；选择优质高β |
| 过热 | 杠杆激增、低质资产热捧、散户涌入 | 减码/防守；增现金 |
| 下行 | 违约攀升、去杠杆、信用收紧 | 保留弹药；等更便宜 |

## 快速检查清单
- **市场情绪**：恐惧 vs 贪婪？机构/散户资金流？
- **信用周期**：利差、违约率、发债难度？
- **估值**：高收益债/股票风险溢价 vs 历史？
- **自身筹码**：现金比重能否撑过极端？
- **时间维度**：投资期够长否？可承受短期帐面亏损？

## 与宏观/流动性的连结
- **Fed 政策转向**：紧 → 松 常伴周期底部，但滞后；信用利差先动。
- **高利率末段**：高收益债相对吸引；注意违约尾巴风险。
- **流动性枯竭**：非流动性资产暴跌是底部讯号之一；须有续投能力。
- **净流动性**：下降 → 风险资产先承压；大幅回升 → 周期反转催化。

## DECISION_RULES（决策规则）

### 周期定位规则
```
IF 信用凍結 + 恐慌拋售 + 估值極低 (底部)
   THEN 積極進場，強調存續力
   BECAUSE 極端後必回歸

IF 利差收窄 + 風險偏好回升 (復甦)
   THEN 持有，選擇優質高β
   BECAUSE 週期上升

IF 槓桿激增 + 低質資產熱捧 + 散戶湧入 (過熱)
   THEN 減碼/防守，增現金
   BECAUSE 週期見頂

IF 違約攀升 + 去槓桿 + 信用收緊 (下行)
   THEN 保留彈藥，等更便宜
   BECAUSE 還沒到底
```

### 逆向规则
```
IF 恐懼指數極高 + 估值極低
   THEN 考慮買入
   BECAUSE 別人恐懼時貪婪

IF 貪婪指數極高 + 估值極高
   THEN 考慮減倉
   BECAUSE 別人貪婪時恐懼

IF 共識認為「這次不一樣」
   THEN 提高警惕
   BECAUSE 週期規律不變
```

### 风险规则
```
IF 下檔保護不足
   THEN 不買
   BECAUSE 追求非對稱回報

IF 最壞情況會導致永久虧損
   THEN 不買或減倉
   BECAUSE 風險 ≠ 波動

IF 自身現金比重無法撐過極端
   THEN 減少暴露
   BECAUSE 生存優先
```

### 信用周期规则
```
IF 利差過窄（歷史低位）
   THEN 減少信用暴露
   BECAUSE 風險定價不足

IF 利差極寬（歷史高位）AND 非系統性危機
   THEN 增加信用暴露
   BECAUSE 風險溢價過高

IF 違約率開始攀升
   THEN 提高質量要求
   BECAUSE 週期轉向
```
