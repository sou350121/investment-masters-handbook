# Howard Marks 投資框架

## QUICK_FACTS
- **基金**：Oaktree Capital
- **規模**：$190B
- **核心風格**：週期意識 + 第二層思考 + 風險控制
- **最佳適用**：週期定位、信用週期、逆向時機

> 重點：週期、風險控制、第二層思考；「當下處於週期的哪裡？」是永恆問題。

## 核心原則
1. **週期意識**：經濟、信用、情緒皆呈週期；極端後必回歸。
2. **第二層思考**：共識已被定價，超額收益來自「共識是錯的」或「定價不足」。
3. **風險 ≠ 波動**：真正風險是永久虧損；波動只是價格噪音。
4. **非對稱回報**：下檔保護優先，有限損失無限上漲。
5. **逆向 + 紀律**：群眾恐慌時買、瘋狂時賣，但需有分析支持。

## 週期定位框架
| 階段 | 特徵 | 行動 |
|------|------|------|
| 底部 | 信用凍結、恐慌拋售、估值極低 | 積極進場；強調存續力 |
| 復甦 | 利差收窄、風險偏好回升 | 持有；選擇優質高β |
| 過熱 | 槓桿激增、低質資產熱捧、散戶湧入 | 減碼/防守；增現金 |
| 下行 | 違約攀升、去槓桿、信用收緊 | 保留彈藥；等更便宜 |

## 快速檢查清單
- **市場情緒**：恐懼 vs 貪婪？機構/散戶資金流？
- **信用週期**：利差、違約率、發債難度？
- **估值**：高收益債/股票風險溢價 vs 歷史？
- **自身籌碼**：現金比重能否撐過極端？
- **時間維度**：投資期夠長否？可承受短期帳面虧損？

## 與宏觀/流動性的連結
- **Fed 政策轉向**：緊 → 鬆 常伴週期底部，但滯後；信用利差先動。
- **高利率末段**：高收益債相對吸引；注意違約尾巴風險。
- **流動性枯竭**：非流動性資產暴跌是底部訊號之一；須有續投能力。
- **淨流動性**：下降 → 風險資產先承壓；大幅回升 → 週期反轉催化。

## DECISION_RULES（決策規則）

### 週期定位規則
```
IF 信用凍結 + 恐慌拋售 + 估值極低 (底部)
   THEN 積極進場，強調存續力
   BECAUSE 極端後必回歸

IF 利差收窄 + 風險偏好回升 (復甦)
   THEN 持有，選擇優質高β
   BECAUSE 週期上升

IF 槓桿激增 + 低質資產熱捧 + 散戶湧入 (過熱)
   THEN 減碼/防守，增現金
   BECAUSE 週期見頂

IF 違約攀升 + 去槓桿 + 信用收緊 (下行)
   THEN 保留彈藥，等更便宜
   BECAUSE 還沒到底
```

### 逆向規則
```
IF 恐懼指數極高 + 估值極低
   THEN 考慮買入
   BECAUSE 別人恐懼時貪婪

IF 貪婪指數極高 + 估值極高
   THEN 考慮減倉
   BECAUSE 別人貪婪時恐懼

IF 共識認為「這次不一樣」
   THEN 提高警惕
   BECAUSE 週期規律不變
```

### 風險規則
```
IF 下檔保護不足
   THEN 不買
   BECAUSE 追求非對稱回報

IF 最壞情況會導致永久虧損
   THEN 不買或減倉
   BECAUSE 風險 ≠ 波動

IF 自身現金比重無法撐過極端
   THEN 減少暴露
   BECAUSE 生存優先
```

### 信用週期規則
```
IF 利差過窄（歷史低位）
   THEN 減少信用暴露
   BECAUSE 風險定價不足

IF 利差極寬（歷史高位）AND 非系統性危機
   THEN 增加信用暴露
   BECAUSE 風險溢價過高

IF 違約率開始攀升
   THEN 提高質量要求
   BECAUSE 週期轉向
```
