# George Soros 投資框架

## QUICK_FACTS
- **基金**：Quantum Fund（1973-2011 主動管理）
- **規模**：峰值 $25B
- **核心風格**：反身性 + 攻擊失衡 + 生存優先
- **最佳適用**：貨幣交易、宏觀失衡、危機交易

> **風格**：宏觀投機、貨幣攻擊、反身性理論

## 核心原則
1. **反身性（Reflexivity）**：市場參與者的認知影響基本面，基本面又反過來影響認知，形成自我強化/崩潰迴圈。
2. **找失衡**：尋找認知與現實的巨大落差；當落差不可持續時下重注。
3. **先投資再調查**：建立初始倉位測試假設，對了加碼，錯了快砍。
4. **生存優先**：錯誤時迅速止損；「活著」比「對」重要。
5. **槓桿放大**：在高確信時使用槓桿；但嚴格控制最大回撤。

## 反身性應用框架
| 階段 | 特徵 | 行動 |
|------|------|------|
| 趨勢啟動 | 基本面改善 + 認知開始跟上 | 順勢建倉 |
| 自我強化 | 價格上漲 → 改善基本面 → 更多買盤 | 持有/加碼 |
| 測試極限 | 認知過度樂觀，價格脫離基本面 | 警惕、縮倉 |
| 崩潰 | 現實戳破認知，拋售潮 | 反向做空或等待 |

## 快速檢查清單
- **失衡存在嗎？** 政策/匯率/信用是否明顯不可持續？
- **反身性方向？** 趨勢在自我強化還是即將反轉？
- **催化劑？** 什麼事件會戳破認知？
- **止損點？** 錯了怎麼跑？最大損失可接受嗎？
- **槓桿水位？** 倉位規模與確信度匹配嗎？

## 與宏觀/流動性的連結
- **固定匯率制**：當央行耗盡外儲，匯率脫鉤是高勝率機會。
- **信用泡沫**：槓桿 → 資產價格 → 更多槓桿；直到流動性收緊。
- **央行政策分歧**：利差 + 資本流動 → 貨幣趨勢。
- **危機傳染**：一個市場崩潰 → 流動性抽離 → 傳染其他新興市場。

## DECISION_RULES（決策規則）

### 反身性交易規則
```
IF 趨勢啟動（基本面改善 + 認知跟上）
   THEN 順勢建倉
   BECAUSE 反身性開始自我強化

IF 自我強化階段（價格 → 基本面 → 更多買盤）
   THEN 持有/加碼
   BECAUSE 趨勢會超預期

IF 測試極限（認知過度樂觀，價格脫離基本面）
   THEN 警惕、縮倉
   BECAUSE 接近反轉點

IF 崩潰階段（現實戳破認知）
   THEN 反向做空或等待
   BECAUSE 反身性向下
```

### 失衡交易規則
```
IF 固定匯率制 AND 外儲耗竭 AND 經濟失衡
   THEN 做空該貨幣
   BECAUSE 不可持續必將崩潰

IF 政策/匯率明顯不可持續 AND 有催化劑
   THEN 重注
   BECAUSE 高勝率機會

IF 信用泡沫（槓桿 → 資產價格 → 更多槓桿）AND 流動性開始收緊
   THEN 準備做空
   BECAUSE 泡沫必破
```

### 生存規則
```
IF 論點被證偽 OR 虧損超過預設止損
   THEN 立即止損
   BECAUSE 活著比對重要

IF 錯了
   THEN 快砍，不等
   BECAUSE 錯誤時迅速止損

IF 確信度不夠
   THEN 小倉位測試
   BECAUSE 先投資再調查
```

### 倉位規則
```
IF 高確信 + 失衡明顯 + 有催化劑
   THEN 可用槓桿重倉
   BECAUSE 難得機會

IF 測試假設階段
   THEN 小倉位
   BECAUSE 對了加碼，錯了快砍

IF 最大回撤 > 容忍度
   THEN 減倉
   BECAUSE 嚴格控制回撤
```
