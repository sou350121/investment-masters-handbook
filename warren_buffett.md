# Warren Buffett 投資框架

## QUICK_FACTS
- **基金**：Berkshire Hathaway
- **規模**：$900B+
- **核心風格**：護城河 + 安全邊際 + 長期持有
- **最佳適用**：選股估值、護城河評估、資本配置

> 聚焦「企業為債、股東為債權人」的現金流視角；強調護城河、資本配置與估值安全邊際。

## 核心原則
- 企業如同債券：看「現金流票息」與「成長率」，折現決定價值。
- 护城河與競爭優勢：品牌/網路效應/成本優勢/轉換成本。
- 資本配置紀律：再投資回報率 > 要求報酬率；若否，回購/分紅。
- 槓桿謹慎：避免永久性損失風險；保留充足現金對衰退/撿便宜。
- 估值安全邊際：好公司也需合理價；偏好可預測現金流、抗通膨能力。

## 快速檢查清單
- 競爭力：市佔/毛利穩定；ROIC 高於 WACC 且持續。
- 現金流質量：FCF/淨利比、營運現金流占利潤、存貨/應收周轉。
- 資本配置：再投資回報率、回購是否在合理價、負債期限/成本。
- 估值：收益率 vs 長端利率（或自定貼現率）；成長假設是否保守。
- 風險：關鍵依賴（單一產品/供應/監管）；周期敏感度。

## 與宏觀的連結
- 實質利率上行 → 估值折現率上升，安全邊際要求更高；偏好高現金轉化、低槓桿。
- 流動性收縮/衰退期：持現金等待撿便宜；寧願錯過、不冒永久損失。
- 通膨環境：看定價權、成本轉嫁力；壽命長的品牌/網路型護城河較抗侵蝕。

## 如何判斷低估（Buffett 視角）

### 🎯 內在價值估算框架

**1. 現金流折現（Owner Earnings）**
```
內在價值 = Σ [未來自由現金流 ÷ (1 + 貼現率)^t]

Owner Earnings = 淨利
                + 折舊/攤銷
                - 維持性資本支出
                - 營運資本增加
```

**關鍵問題**：
- ✅ 未來 10 年現金流可預測嗎？（護城河夠寬？）
- ✅ 維持性資本支出是多少？（警惕低報的公司）
- ✅ 營運資本是否持續膨脹？（存貨、應收堆積）
- ✅ 貼現率用多少？（10Y 國債 + 4-6% 風險溢價）

**2. 盈餘收益率法（簡化版）**
```
盈餘收益率 = Owner Earnings ÷ 市值
對比：10Y 國債收益率

若 盈餘收益率 > 國債收益率 × 1.5 → 可能低估
若 盈餘收益率 < 國債收益率 → 可能高估
```

**3. 相對估值（謹慎使用）**
- P/E vs 歷史區間：但**只在護城河未變時有效**
- P/B 適用於金融/重資產：ROE 穩定時可用
- EV/EBITDA：適合比較資本結構不同的公司

**Buffett 特別提醒**：
> "估值不是科學，是藝術。但藝術也需要紀律。"
- 不要為了買而降低標準
- 寧可以合理價買優秀公司，也不以便宜價買平庸公司

### 🔍 低估的信號（逐項檢查）

| 指標 | 低估信號 | 數字標準 | 陷阱警示 |
|------|----------|----------|----------|
| **盈餘收益率** | > 10Y 國債 × 1.5 | 假設國債 4% → 需 > 6% | 盈餘品質？一次性項目？ |
| **FCF Yield** | > 8-10% | FCF/市值 > 8% | 維持性支出被低估？ |
| **P/E** | < 歷史均值 - 1σ | 需確認護城河未變 | 盈利見頂？週期高點？ |
| **P/B** | < 1.2 且 ROE > 15% | 金融股/重資產適用 | 資產減值？不良資產？ |
| **PEG** | < 0.8 | P/E ÷ 增速 | 增速不可持續？ |
| **EV/EBIT** | < 10 | 好生意：8-12 | 槓桿過高掩蓋？ |
| **股息收益率** | > 國債 + 2% | 假設國債 4% → 需 > 6% | 股息不可持續？ |

### 💎 低估的定性信號

**市場情緒面**
- ✅ 該行業/公司被「暫時」冷落（非結構性衰退）
- ✅ 短期負面新聞（訴訟、召回、季報不佳）但長期不受影響
- ✅ 「無聊」的生意：沒有成長故事，分析師不關注
- ✅ 指數調整/被動基金被迫賣出（非基本面原因）

**Buffett 經典案例**
| 公司 | 年份 | 低估原因 | 買入邏輯 |
|------|------|----------|----------|
| American Express | 1963 | 色拉油醜聞，股價腰斬 | 品牌護城河未損，暫時危機 |
| 華盛頓郵報 | 1973 | 市場崩盤，媒體被錯殺 | 區域壟斷地位，市值 < 清算價 |
| 富國銀行 | 1990 | 房地產危機恐慌 | 管理優秀，危機中的錯殺 |
| 可口可樂 | 1988 | 市場認為成長停滯 | 全球品牌價值被低估 |

**低估的必要條件（缺一不可）**
1. 護城河依然存在（競爭優勢未被破壞）
2. 管理層誠信且有能力
3. 財務健康（低槓桿、現金流正）
4. 有安全邊際（市價 < 內在價值的 60-70%）

### 🔥 高估的市場情緒信號

**狂熱跡象（Buffett 多次提醒）**

- ✅ 估值用「新指標」合理化（如用戶數、點擊率代替盈利）

- ✅ 「股神已老」、「價值投資已死」成為共識
- ✅ 出租車司機/理髮師都在談論股票

> "別人貪婪時恐懼，別人恐懼時貪婪。"
> "當市場狂熱時，我選擇去打橋牌。"

### 📊 高估的行業/公司特徵

**警示信號**
- [ ] **估值需要「信仰」**：需相信顛覆性技術、指數級增長
- [ ] **盈利遙遙無期**：「規模化後就盈利」的承諾
- [ ] **會計激進**：確認收入過早、資本化過多、一次性調整頻繁
- [ ] **股權稀釋嚴重**：員工期權、可轉債不斷稀釋股東
- [ ] **自由現金流為負**：持續燒錢、依賴融資
- [ ] **併購狂魔**：靠併購做大營收、掩蓋內生增長乏力
- [ ] **管理層高調**：頻繁路演、承諾宏大願景、股票質押

**Buffett 的謹慎**
| 情境 | Buffett 做法 | 為什麼 |
|------|-------------|--------|
| 科技股泡沫（1999-2000） | 持有現金、不參與 | "不懂就不碰" |
| 加密貨幣熱潮（2017-2021） | 公開批評「無價值」 | "不產生現金流" |
| SPAC 熱潮（2020-2021） | 遠離 | "結構性激勵錯位" |

### 🛡️ 避免高估的決策紀律

**1. 機會成本思維**
```
每個決策都在和「持有現金」或「國債」競爭

問自己：
- 這筆投資的預期回報 > 10-12%（長期股市均值）嗎？
- 風險調整後，真的比國債/現金好嗎？
- 如果用同樣的錢等 2 年，會有更好的機會嗎？
```

**2. 反向壓力測試**
- 假設盈利增速減半，估值還合理嗎？
- 假設利率上升 2%，貼現率對估值的影響？
- 假設護城河被侵蝕 20%，市價是否還有支撐？

**3. Buffett 的「打孔卡」理念**
> "假設你一生只有 20 個投資決策的機會（20 個打孔位），你還會買這個嗎？"

**4. 等待「肥球」（Fat Pitch）**
- 不是每個球都要揮棒
- 等待「護城河強 + 管理層優秀 + 估值便宜」三者合一的機會
- 寧可錯過，不要犯錯

## DECISION_RULES（決策規則）

### 買入規則
```
IF 盈餘收益率 > 10Y國債 × 1.5 AND 護城河完好
   THEN 考慮買入
   BECAUSE 提供安全邊際，回報優於無風險利率

IF FCF Yield > 8% AND ROIC > 15% 持續 5 年+
   THEN 深入研究
   BECAUSE 可能是複利機器

IF 短期負面新聞 AND 護城河未損 AND 股價下跌 > 30%
   THEN 機會窗口
   BECAUSE 市場過度反應暫時性問題

IF P/E < 歷史均值 - 1σ AND 護城河未變 AND 管理層誠信
   THEN 考慮買入
   BECAUSE 週期性低估

IF 「無聊」的生意 AND 分析師不關注 AND FCF 穩定
   THEN 深入研究
   BECAUSE 可能是隱藏的好公司
```

### 賣出規則
```
IF 護城河被侵蝕 AND 無改善跡象
   THEN 賣出
   BECAUSE 競爭優勢消失

IF 估值 > 內在價值 × 1.5
   THEN 考慮減倉
   BECAUSE 安全邊際消失

IF 管理層誠信問題
   THEN 立即賣出
   BECAUSE 無法信任

IF 有更好的機會 AND 當前持倉估值合理但不便宜
   THEN 換倉
   BECAUSE 機會成本
```

### 不買規則
```
IF 無法用 2 句話解釋商業模式
   THEN 不買
   BECAUSE 能力圈外

IF 需要「信仰」未來才能合理化估值
   THEN 不買
   BECAUSE 投機而非投資

IF FCF 長期為負 AND 無明確盈利路徑
   THEN 不買
   BECAUSE 燒錢模式

IF 高槓桿 AND 短期債務 > 現金
   THEN 不買
   BECAUSE 衰退時破產風險

IF 管理層頻繁質押股票 OR 高調承諾
   THEN 不買
   BECAUSE 激勵錯位信號
```

## 估值小工具（快速計算）

### 工具 1：Owner Earnings 快算
```python
# 步驟 1：拿到財報數據
淨利 = 100
折舊攤銷 = 20
維持性資本支出 = 15  # 經驗：約折舊的 70-100%
營運資本增加 = 5

# 步驟 2：計算
Owner_Earnings = 淨利 + 折舊攤銷 - 維持性資本支出 - 營運資本增加
                = 100 + 20 - 15 - 5 = 100

# 步驟 3：估值
市值 = 1000
盈餘收益率 = 100 / 1000 = 10%

# 步驟 4：對比
國債收益率 = 4%
要求收益率 = 6%（國債 × 1.5）

結論：10% > 6%，可能低估（需進一步確認護城河）
```

### 工具 2：兩階段 DCF（保守假設）
```
假設條件：
- 前 5 年增速：8%
- 永續增速：3%（接近 GDP）
- 貼現率：10%（國債 4% + 6% 風險溢價）

當前 FCF = 100
市值 = 1000

計算：
階段 1（前 5 年）折現值 = 100 × [(1+8%)/(1+10%) + ... ] ≈ 416
階段 2（永續）折現值 = 100 × (1+8%)^5 × (1+3%)/(10%-3%) / (1+10%)^5 ≈ 872

內在價值 = 416 + 872 = 1288
安全邊際 = (1288 - 1000) / 1288 = 22%

結論：安全邊際 > 20%，可考慮（Buffett 偏好 > 30%）
```
