## 二、案例一：如何用这套框架分析一只股票

### 📌 案例：分析 Costco (COST)

假设你正在考虑是否买入 Costco，以下是完整分析流程：

---

### Step 1：宏观定位（5 分钟）

**参考**：`ray_dalio.md`、`stanley_druckenmiller.md`

**问自己**：
- 当前经济象限？（增长↑↓ × 通膨↑↓）
- 流动性趋势？（Fed B/S − TGA − RRP）
- 利率环境？

**应用 IF-THEN 规则**：
```
當前假設：增長↓ + 通膨↓ (衰退/通縮風險)

IF 增長↓ + 通膨↓
   THEN 偏防守，消費必需品（如 Costco）相對有利
   BECAUSE 衰退環境中必需消費較抗跌
```

**结论**：宏观环境对 Costco（必需消费）中性偏正面 ✅

---

### Step 2：护城河评估（15 分钟）

**参考**：`warren_buffett.md` 的护城河框架

**Buffett 护城河检查清单**：

| 护城河类型 | Costco 是否具备 | 评估 |
|-----------|----------------|------|
| 品牌 | ✅ 会员信任度极高 | 强 |
| 网络效应 | ⚠️ 有限 | 中 |
| 成本优势 | ✅ 规模采购、低毛利策略 | 强 |
| 转换成本 | ✅ 会员制 + 习惯 | 中强 |

**应用 IF-THEN 规则**：
```
IF 品牌/成本優勢明確 AND ROIC > WACC 持續 5年+
   THEN 護城河可能存在
   
Costco: ROIC ~20%+ 持續多年 > WACC ~8%
結論：護城河存在 ✅
```

---

### Step 3：估值判断（15 分钟）

**参考**：`warren_buffett.md`、`peter_lynch.md`

**Buffett 估值框架**：

```
當前數據（假設）：
- 股價：$800
- EPS：$15
- P/E：53x
- 10Y 國債：4.5%
- 盈餘收益率：1.9%（= 1/53）

應用規則：
IF 盈餘收益率 > 10Y國債 × 1.5
   盈餘收益率 1.9% vs 國債×1.5 = 6.75%
   1.9% < 6.75%
   THEN 以 Buffett 標準，估值偏高 ⚠️
```

**Lynch PEG 框架**：

```
假設：
- P/E：53x
- 預期 EPS 增速：12%
- PEG = 53 / 12 = 4.4

應用規則：
IF PEG > 2
   THEN 以 Lynch 標準，估值偏高 ⚠️
```

**结论**：估值偏高，安全边际不足 ⚠️

---

### Step 4：决策品质检查（5 分钟）

**参考**：`charlie_munger.md` 的心理偏误清单

**Munger 检查清单**：

| 偏误 | 自问 | 回答 |
|------|------|------|
| 确认偏误 | 我只看到利好了吗？ | 需要找估值偏高的反驳论点 |
| 从众效应 | 大家都在买吗？ | Costco 是热门股，需警惕 |
| 锚定效应 | 被之前的价格锚定了吗？ | 忘记历史价格，重新评估 |

**应用 IF-THEN 规则**：
```
IF 無法用 2 句話解釋商業模式
   THEN 不買（能力圈外）
   
Costco: "會員制倉儲超市，靠會員費盈利，商品低毛利高周轉"
→ 2 句話可解釋 ✅，在能力圈內
```

---

### Step 5：综合决策

**汇总**：

| 维度 | 评估 | 结论 |
|------|------|------|
| 宏观环境 | 衰退风险，必需消费有利 | ✅ 正面 |
| 护城河 | 品牌 + 成本优势 + 会员制 | ✅ 存在 |
| 估值 | P/E 53x, PEG 4.4 | ⚠️ 偏高 |
| 能力圈 | 商业模式清晰 | ✅ 理解 |

**最终决策（应用 Buffett 规则）**：

```
IF 護城河完好 BUT 安全邊際 < 30%
   THEN 等待更好價格
   BECAUSE 好公司也需合理價

結論：
- Costco 是好公司 ✅
- 當前估值偏高 ⚠️
- 行動：加入觀察名單，等待回調（如 P/E < 35）再考慮
```

## 三、案例二：如何判断当前市场周期位置

### 📌 案例：2024 年 Q4 市场周期判断

---

### Step 1：经济四象限定位（Dalio）

**参考**：`ray_dalio.md`

**收集数据**：

| 指标 | 当前值 | 趋势 |
|------|--------|------|
| GDP 增速 | 2.5% | ↓ 放缓 |
| CPI YoY | 3.2% | ↓ 回落 |
| 核心 PCE | 2.8% | ↓ 回落 |
| 失业率 | 4.1% | ↑ 上升 |

**应用 IF-THEN 规则**：
```
IF 增長↓ + 通膨↓
   THEN 處於「衰退/通縮」象限邊緣
   偏好資產：長期國債、現金、防禦股
```

---

### Step 2：流动性趋势（Druckenmiller）

**参考**：`stanley_druckenmiller.md`

**计算净流动性**：
```
淨流動性 = Fed B/S − TGA − RRP

假設數據：
- Fed B/S：$7.5T
- TGA：$0.8T
- RRP：$0.5T
- 淨流動性：$6.2T

3 個月變化：+$0.3T（上升）
```

**应用 IF-THEN 规则**：
```
IF 淨流動性上升 AND 估值合理
   THEN 做多風險資產
   BECAUSE 流動性推動市場

當前：淨流動性上升 ✅，但估值偏高 ⚠️
→ 謹慎樂觀
```

---

### Step 3：周期情绪判断（Marks）

**参考**：`howard_marks.md`

**周期检查清单**：

| 指标 | 当前状态 | 周期信号 |
|------|---------|---------|
| 信用利差（HY-IG） | 偏窄 | ⚠️ 风险定价不足 |
| VIX | 15-18 | ⚠️ 自满 |
| 散户情绪 | 偏乐观 | ⚠️ 偏热 |
| IPO 市场 | 活跃 | ⚠️ 偏热 |
| 杠杆水平 | 中等 | ✅ 尚可 |

**应用 IF-THEN 规则**：
```
IF 信用利差偏窄 + 散戶樂觀 + VIX 低
   THEN 週期偏熱，但未到極端
   BECAUSE 風險定價不足是警示信號

IF 「這次不一樣」成為共識
   THEN 提高警惕
   當前：尚未到此程度
```

---

### Step 4：综合周期定位

**Marks 周期框架对照**：

| 阶段 | 特征 | 当前匹配度 |
|------|------|-----------|
| 底部 | 恐慌、信用冻结、估值极低 | ❌ |
| 复苏 | 利差收窄、风险偏好回升 | ⚠️ 部分 |
| **过热** | 杠杆增、低质资产热捧 | ⚠️ 部分特征 |
| 下行 | 违约升、信用收紧 | ❌ |

**结论**：
```
當前週期位置：復甦後期 → 過熱早期
建議行動：
- 不追高
- 維持部分現金（20-30%）
- 關注信用利差變化
- 準備好應對潛在回調
```

---

## 四、案例三：如何设计 AI 投资助手的 System Prompt

### 📌 三种 Prompt 设计方案

---

### 方案 A：极简版（直接使用）

适用：快速部署、Token 限制严格

```markdown
# System Prompt - 投資決策助手（極簡版）

你是一位投資決策助手，根據以下框架回答：

## 核心投資人參考
- 選股：Warren Buffett（護城河+安全邊際）、Peter Lynch（PEG<1）
- 宏觀：Ray Dalio（四象限）、Stanley Druckenmiller（流動性）
- 風控：Charlie Munger（避免偏誤）、Howard Marks（週期）

## 決策規則
- 買入：盈餘收益率 > 國債×1.5 AND 護城河OK
- 賣出：護城河被侵蝕 OR 論點不成立
- 倉位：確信度90%+ → 重倉，<50% → 不做

## 回答格式
1. 問題類型識別
2. 參考投資人
3. 應用 IF-THEN 規則
4. 給出建議
```

---

### 方案 B：标准版（推荐）

适用：一般 AI 助手，平衡深度与 Token

```markdown
# System Prompt - 投資決策助手（標準版）

你是一位專業投資決策助手，整合多位傳奇投資人的智慧框架。

## 角色定位
- 提供投資決策參考，而非具體買賣建議
- 強調框架思維，而非預測市場
- 幫助用戶避免常見錯誤

## 投資人快速參考

| 問題類型 | 參考投資人 | 核心框架 |
|----------|-----------|---------|
| 選股估值 | Buffett | 盈餘收益率 > 國債×1.5，護城河 |
| 成長股 | Lynch | PEG < 1，增速可持續 |
| 深度價值 | Klarman | 安全邊際 > 30%，冷門資產 |
| 宏觀週期 | Dalio | 增長×通膨四象限 |
| 流動性 | Druckenmiller | 淨流動性 = Fed B/S − TGA − RRP |
| 週期判斷 | Marks | 恐懼/貪婪極端，信用週期 |
| 決策檢查 | Munger | 心理偏誤，能力圈 |
| 止損 | Soros | 論點錯了就跑 |

## 核心 IF-THEN 規則

### 買入
```
IF 盈余收益率 > 10Y国债 × 1.5 AND 护城河完好
   THEN 考虑买入 (Buffett)
   
IF PEG < 1 AND 增速可持续 5年+
   THEN 成长股机会 (Lynch)
   
IF 安全边际 > 30% AND 有催化剂
   THEN 考虑买入 (Klarman)
```

### 賣出
```
IF 护城河被侵蚀 AND 无改善迹象
   THEN 卖出 (Buffett)
   
IF 买入论点不再成立
   THEN 立即止损 (Soros)
```

### 倉位
```
IF 确信度 90%+ AND 论点清晰
   THEN 重仓 10-25% (Druckenmiller)
   
IF 确信度 < 50%
   THEN 小仓位或不做
```

### 心理檢查
```
IF 只看到利好信息 → 确认偏误，找反驳论点
IF 「大家都在买」 → 从众效应，独立思考
IF 被成本价锚定 → 忘记成本，重新评估
```

## 回答結構

對於投資相關問題，按以下結構回答：

1. **問題識別**：這是選股/宏觀/風控問題？
2. **框架選擇**：參考哪位投資人的框架？
3. **規則應用**：應用相關 IF-THEN 規則
4. **風險提示**：指出潛在偏誤或風險
5. **行動建議**：具體可執行的下一步

## 注意事項

- 不提供具體股票買賣建議
- 強調「這是框架參考，需要用戶自行判斷」
- 遇到不確定時，引用 Munger：「我不知道」是合理答案
- 提醒用戶做好風險管理
```

### 📋 Prompt 测试案例

**测试 1：选股问题**
```
用戶：我在考慮買入 Apple (AAPL)，你怎麼看？

預期回答結構：
1. 問題類型：選股估值
2. 參考：Buffett（護城河）、Lynch（PEG）
3. 框架應用：
   - 護城河：品牌 ✅、生態系統 ✅、轉換成本 ✅
   - 估值：P/E、盈餘收益率 vs 國債
4. 風險提示：確認偏誤（Apple 是熱門股）
5. 建議：需要具體數據才能判斷估值
```

**测试 2：宏观问题**
```
用戶：現在市場是不是泡沫？

預期回答結構：
1. 問題類型：週期/泡沫判斷
2. 參考：Munger、Marks、Burry
3. 框架應用：
   - Munger 泡沫信號檢查
   - Marks 週期位置
4. 需要用戶提供的數據：估值水平、信用利差、散戶情緒
5. 風險提示：避免預測具體時點
```

**测试 3：心理偏误**
```
用戶：我買的股票跌了 30%，該不該止損？

預期回答結構：
1. 問題類型：止損決策 + 心理檢查
2. 參考：Soros（止損）、Munger（偏誤）
3. 關鍵問題：買入論點還成立嗎？（與價格無關）
4. 偏誤提醒：損失厭惡、錨定效應
5. IF-THEN：論點不成立 → 止損；論點成立 → 可持有
```

## 五、日常使用习惯

### 📅 每日（5 分钟）

```markdown
□ 瀏覽主要指數、利率、美元
□ 有無重大央行/經濟數據？
□ 持倉有無重大新聞？
```

### 📅 每周（30 分钟）

```markdown
□ 更新淨流動性追蹤（Druckenmiller）
  - Fed B/S − TGA − RRP = ?
□ 檢查持倉基本面新聞
□ 信用利差變化（Marks）
```

### 📅 每月（2 小时）

```markdown
□ 經濟四象限重新評估（Dalio）
□ 週期情緒評估（Marks）
□ 持倉論點復盤
□ 觀察名單更新
```

### 📅 每季（半天）

```markdown
□ 完整組合檢視
```

## 附录：快速参考卡片

### 买入决策速查
```
✅ 盈餘收益率 > 國債×1.5？
✅ 護城河存在？（品牌/網絡/成本/轉換成本）
✅ PEG < 1？（成長股）
✅ 安全邊際 > 30%？
✅ 能 2 句話解釋商業模式？
✅ 沒有心理偏誤？
```

### 卖出决策速查
```
⚠️ 護城河被侵蝕？
⚠️ 買入論點不成立？
⚠️ 估值 > 內在價值×1.5？
⚠️ 管理層誠信問題？
⚠️ 有更好機會？
```

### 周期位置速查
```
底部信號：恐慌拋售、信用凍結、估值極低
過熱信號：槓桿激增、散戶湧入、「這次不一樣」
```

### 心理偏误速查
```
確認偏誤：只看利好 → 找反駁論點
從眾效應：大家都買 → 獨立思考
損失厭惡：虧損不賣 → 論點對不對？
錨定效應：被成本錨定 → 忘記成本
```
