# 版本策略（Versioning）

> 本文档用于说明本仓库的版本号规范、发布节奏与兼容性承诺。
> 该文件用于修复文档导航中的坏链接，并作为后续完善的占位符（见 `docs/DEVELOPMENT_PLAN.md`）。

## 约定（草案）

- **语义化版本（SemVer）**：`MAJOR.MINOR.PATCH`
  - **MAJOR**：不向后兼容的变更
  - **MINOR**：向后兼容的新功能
  - **PATCH**：向后兼容的 bugfix / 文档修复

## 发布说明

- 变更记录请参考：`CHANGELOG.md` 与 `docs/RELEASE_NOTES.md`
