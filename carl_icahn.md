# Carl Icahn 投資框架

## QUICK_FACTS
- **基金**：Icahn Enterprises（NYSE: IEP）
- **規模**：$15B+
- **核心風格**：激進股東 + 治理套利 + 價值釋放
- **最佳適用**：激進投資、公司治理改善、解鎖隱藏價值

> **風格**：激進股東、公司治理套利、解鎖隱藏價值

## 核心原則
1. **股東權益優先**：管理層是代理人；若不為股東創造價值，就該換掉。
2. **找治理缺陷**：低效管理、過度薪酬、資本配置失誤 → 介入空間。
3. **累積影響力**：買到足夠股份進董事會或施壓。
4. **催化價值釋放**：分拆、回購、出售資產、更換管理層。
5. **耐心對峙**：與管理層談判/對抗可能持續數年。

## 激進投資流程
| 階段 | 行動 |
|------|------|
| 1. 篩選 | 找低估值 + 治理問題 + 可改善空間 |
| 2. 建倉 | 逐步累積至 5-15%（觸發 13D 披露） |
| 3. 介入 | 公開信、提名董事、股東提案 |
| 4. 談判 | 與管理層/董事會協商改革方案 |
| 5. 退出 | 價值釋放後賣出；或被收購 |

## 經典戰役
| 年份 | 公司 | 結果 |
|------|------|------|
| 1985 | TWA | 收購後重組 |
| 2013 | Apple | 推動回購（成功） |
| 2013 | Dell | 反對私有化（最終失敗但獲利） |
| 2015 | Xerox | 推動分拆 |
| 2020 | Occidental | 反對 Anadarko 收購（部分成功） |

## 快速檢查清單
- **估值折價？** 股價 vs 資產價值/同業？
- **治理問題？** 管理層薪酬、資本配置、董事會結構？
- **改善空間？** 分拆、回購、出售、成本削減？
- **股東結構？** 是否有其他激進股東？機構支持？
- **法律/章程障礙？** 毒丸、分級董事會、超級投票權？
- **退出路徑？** 誰會收購？或市場重新定價？

## 與宏觀/週期的連結
- **低利率**：便宜融資支持槓桿收購；激進活動活躍。
- **市場下跌**：便宜買入低估標的；管理層壓力增大。
- **併購週期**：活躍併購市場提供退出渠道。
- **經濟衰退**：困境公司增加，但融資困難。

## DECISION_RULES（決策規則）

### 篩選規則
```
IF 估值折價（股價 vs 資產/同業）AND 治理問題
   THEN 深入研究
   BECAUSE 激進投資標的

IF 管理層薪酬過高 OR 資本配置失誤
   THEN 有介入空間
   BECAUSE 治理改善潛力

IF 可分拆/回購/出售資產釋放價值
   THEN 有催化劑潛力
   BECAUSE 價值釋放路徑

IF 無毒丸/分級董事會/超級投票權
   THEN 法律障礙低
   BECAUSE 可執行性高
```

### 建倉規則
```
IF 篩選通過 AND 有介入空間
   THEN 逐步累積至 5-15%
   BECAUSE 達到 13D 披露門檻後施壓

IF 累積足夠股份
   THEN 進入介入階段
   BECAUSE 有影響力後才有效
```

### 介入規則
```
IF 建倉完成
   THEN 公開信 + 提名董事 + 股東提案
   BECAUSE 公開壓力

IF 管理層抵抗
   THEN 代理權戰爭（proxy fight）
   BECAUSE 通過股東投票決定

IF 談判進展
   THEN 協商具體改革方案
   BECAUSE 合作優於對抗
```

### 退出規則
```
IF 價值釋放（分拆/回購/出售/被收購）
   THEN 賣出
   BECAUSE 論點兌現

IF 對抗失敗 AND 無進展
   THEN 重新評估是否退出
   BECAUSE 時間成本

IF 有更好機會
   THEN 可以換倉
   BECAUSE 機會成本
```
