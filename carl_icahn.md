# Carl Icahn 投资框架

## QUICK_FACTS
- **基金**：Icahn Enterprises（NYSE: IEP）
- **规模**：$15B+
- **核心风格**：激进股东 + 治理套利 + 价值释放
- **最佳适用**：激进投资、公司治理改善、解锁隐藏价值

> **风格**：激进股东、公司治理套利、解锁隐藏价值

## 核心原则
1. **股东权益优先**：管理层是代理人；若不为股东创造价值，就该换掉。
2. **找治理缺陷**：低效管理、过度薪酬、资本配置失误 → 介入空间。
3. **累积影响力**：买到足够股份进董事会或施压。
4. **催化价值释放**：分拆、回购、出售资产、更换管理层。
5. **耐心对峙**：与管理层谈判/对抗可能持续数年。

## 激进投资流程
| 阶段 | 行动 |
|------|------|
| 1. 筛选 | 找低估值 + 治理问题 + 可改善空间 |
| 2. 建仓 | 逐步累积至 5-15%（触发 13D 披露） |
| 3. 介入 | 公开信、提名董事、股东提案 |
| 4. 谈判 | 与管理层/董事会协商改革方案 |
| 5. 退出 | 价值释放后卖出；或被收购 |

## 经典战役
| 年份 | 公司 | 结果 |
|------|------|------|
| 1985 | TWA | 收购后重组 |
| 2013 | Apple | 推动回购（成功） |
| 2013 | Dell | 反对私有化（最终失败但获利） |
| 2015 | Xerox | 推动分拆 |
| 2020 | Occidental | 反对 Anadarko 收购（部分成功） |

## 快速检查清单
- **估值折价？** 股价 vs 资产价值/同业？
- **治理问题？** 管理层薪酬、资本配置、董事会结构？
- **改善空间？** 分拆、回购、出售、成本削减？
- **股东结构？** 是否有其他激进股东？机构支持？
- **法律/章程障碍？** 毒丸、分级董事会、超级投票权？
- **退出路径？** 谁会收购？或市场重新定价？

## 与宏观/周期的连结
- **低利率**：便宜融资支持杠杆收购；激进活动活跃。
- **市场下跌**：便宜买入低估标的；管理层压力增大。
- **并购周期**：活跃并购市场提供退出渠道。
- **经济衰退**：困境公司增加，但融资困难。

## DECISION_RULES（决策规则）

### 筛选规则
```
IF 估值折價（股價 vs 資產/同業）AND 治理問題
   THEN 深入研究
   BECAUSE 激進投資標的

IF 管理層薪酬過高 OR 資本配置失誤
   THEN 有介入空間
   BECAUSE 治理改善潛力

IF 可分拆/回購/出售資產釋放價值
   THEN 有催化劑潛力
   BECAUSE 價值釋放路徑

IF 無毒丸/分級董事會/超級投票權
   THEN 法律障礙低
   BECAUSE 可執行性高
```

### 建仓规则
```
IF 篩選通過 AND 有介入空間
   THEN 逐步累積至 5-15%
   BECAUSE 達到 13D 披露門檻後施壓

IF 累積足夠股份
   THEN 進入介入階段
   BECAUSE 有影響力後才有效
```

### 介入规则
```
IF 建倉完成
   THEN 公開信 + 提名董事 + 股東提案
   BECAUSE 公開壓力

IF 管理層抵抗
   THEN 代理權戰爭（proxy fight）
   BECAUSE 通過股東投票決定

IF 談判進展
   THEN 協商具體改革方案
   BECAUSE 合作優於對抗
```

### 退出规则
```
IF 價值釋放（分拆/回購/出售/被收購）
   THEN 賣出
   BECAUSE 論點兌現

IF 對抗失敗 AND 無進展
   THEN 重新評估是否退出
   BECAUSE 時間成本

IF 有更好機會
   THEN 可以換倉
   BECAUSE 機會成本
```
