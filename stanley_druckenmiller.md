# Stanley Druckenmiller 投資框架

## QUICK_FACTS
- **基金**：Duquesne Capital (已關閉) / Quantum Fund
- **規模**：峰值 $12B
- **核心風格**：流動性決定一切 + 集中重注 + 靈活反轉
- **最佳適用**：宏觀擇時、流動性追蹤、Fed 政策判斷

> 重點：宏觀擇時、流動性追蹤、集中下注；「保持靈活並在確定時下重注」。

## 核心原則
1. **宏觀優先**：經濟動能、央行政策、流動性是主線；個股服務於宏觀方向。
2. **集中 & 重注**：看對方向時不怕重倉；小倉位意味「還沒想清楚」。
3. **靈活反轉**：錯了快認錯；盈利也能平倉；無沉沒成本依戀。
4. **流動性至上**：「流動性推動市場，而非基本面」；關注 Fed 資產負債表、TGA、逆回購。
5. **技術面輔助**：用價格確認宏觀判斷；趨勢、成交量、市場寬度是驗證工具。

## 宏觀判斷框架
| 維度 | 觀測指標 | 行動 |
|------|----------|------|
| 流動性 | Fed B/S、淨流動性、M2 增速 | 上升 → 做多風險；下降 → 減倉或做空 |
| 經濟週期 | ISM、就業、消費信心 | 上升期做多高β，衰退期偏防守 |
| 利率預期 | FF 期貨、SOFR-OIS、實質利率 | 轉鴿 → 先入股 → 等信用追趕 |
| 美元強弱 | DXY、新興貨幣壓力 | 強美元 → 新興、商品承壓 |

## 快速檢查清單
- **方向性確信**：能否用一句話講清邏輯？若需多重假設 → 減倉。
- **流動性趨勢**：當前淨流動性（Fed B/S − TGA − RRP）方向？3 個月變化？
- **市場寬度**：多少比例股票 > 200MA？若方向做多但寬度惡化 → 警惕。
- **止損/風控**：進場前設好退出點；錯誤時能否快速平倉？
- **機會成本**：這筆交易的預期報酬 vs. 現金或更好機會？

## 與宏觀/流動性的連結
- **QT → QE 轉換**：Druckenmiller 典型「聞到 Fed 轉向」先佈局成長股/風險資產。
- **高利率尾聲**：通常在最後一次加息前後做多債券，等待降息週期。
- **美元趨勢**：強美元時減少新興、商品；弱美元時增加。
- **危機時刻**：流動性枯竭 → 可能出現超額機會（1992 英鎊、2008 美股、2020 疫情）。

## DECISION_RULES（決策規則）

### 做多規則
```
IF 淨流動性（Fed B/S − TGA − RRP）上升 AND 估值合理
   THEN 做多風險資產
   BECAUSE 流動性推動市場

IF Fed 轉向（緊→鬆）信號確認
   THEN 先佈局成長股
   BECAUSE 流動性改善先反映在股市

IF 確信度 90%+ AND 邏輯一句話講清楚
   THEN 重倉 10-25%
   BECAUSE 看對方向要敢下注

IF 最後一次加息確認
   THEN 做多長期國債
   BECAUSE 歷史規律
```

### 做空/減倉規則
```
IF 淨流動性下降 AND 高估值
   THEN 減倉或做空
   BECAUSE 雙殺風險

IF 方向做多 BUT 市場寬度惡化（< 50% 股票 > 200MA）
   THEN 警惕，考慮減倉
   BECAUSE 市場內部轉弱

IF 需要多重假設才能支持邏輯
   THEN 減倉
   BECAUSE 還沒想清楚
```

### 認錯規則
```
IF 論點被證偽 OR 價格確認反向
   THEN 立即止損
   BECAUSE 錯了快認錯

IF 虧損但論點仍成立 AND 流動性未變
   THEN 可以持有/加倉
   BECAUSE 時機問題而非方向問題
```

### 倉位規則
```
IF 確信度 90%+ THEN 重倉 10-25%
IF 確信度 70% THEN 中等 5-10%
IF 確信度 50% THEN 小倉位或不做
```
