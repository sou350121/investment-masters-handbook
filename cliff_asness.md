## 🎯 核心原则

### 原则 1：因子是收益的真正来源

```
「你以為你在買股票，其實你在買因子暴露。」

CAPM 說只有市場風險有溢價，
Fama-French 說還有規模和價值，
Asness 說還有動量，
而且這些因子可以被系統化收割。
```

### 原则 2：价值和动量可以共存

```
傳統觀點：價值投資者看不起動量
Asness 觀點：它們負相關，組合起來更強

價值：買便宜的，賣貴的
動量：買漲的，賣跌的

聽起來矛盾？
但數據證明：價值+動量的組合
比單獨用任何一個都更穩定。
```

### 原则 3：纪律胜过直觉

```
「規則的價值不在於它總是對的，
而在於它讓你避免最差的決策。」

- 系統化執行，消除情緒
- 定期再平衡，而非追漲殺跌
- 長期堅持，即使短期落後
```

### 原则 4：因子会有艰难时期

```
「每個因子都會經歷它的『死亡時刻』，
這正是為什麼它長期有溢價。」

- 價值因子在 2018-2020 被「殺死」了
- 動量因子在 2009 年 3 月被「殺死」了
- 如果因子永遠有效，它就會被套利掉

風險溢價 = 承擔風險的補償
沒有痛苦的時期，就沒有溢價。
```

## 📊 核心因子框架

### 经典四因子

| 因子 | 定义 | 逻辑 |
|------|------|------|
| **价值 (Value)** | 买便宜的（低 P/B、P/E） | 风险溢价 + 行为偏误 |
| **动量 (Momentum)** | 买近期涨的 | 趋势延续 + 反应不足 |
| **规模 (Size)** | 买小盘股 | 流动性溢价 + 信息不对称 |
| **质量 (Quality)** | 买高 ROE、低杠杆 | 稳定性溢价 |

### 价值因子详解

```
定義：
- 低 P/B（市淨率）
- 低 P/E（市盈率）
- 低 EV/EBITDA
- 高股息率

為什麼有效：
1. 風險補償：便宜的公司通常有問題
2. 行為偏誤：投資者過度外推壞消息
3. 均值回歸：極端估值傾向回歸

風險：
- 價值陷阱（便宜有原因）
- 長期落後期（2017-2020）
- 行業集中（金融、能源）
```

### 动量因子详解

```
定義：
- 過去 12 個月收益率（排除最近 1 個月）
- 價格相對強度
- 52 週高點接近度

為什麼有效：
1. 反應不足：投資者對新信息反應慢
2. 趨勢延續：贏家繼續贏
3. 行為偏誤：處置效應、確認偏誤

風險：
- 動量崩潰（2009.03）
- 高換手率
- 擁擠交易
```

### 价值 + 动量组合

```
為什麼組合更好？

- 價值和動量負相關（約 -0.5）
- 當價值失效時，動量往往有效
- 當動量崩潰時，價值往往保護

實際構建：
1. 分別計算價值得分和動量得分
2. 綜合排名選股
3. 或者：50% 價值組合 + 50% 動量組合
```

## 🛠️ AQR 策略类型

### 策略 1：股票多空（Equity Long/Short）

```
方法：
- 做多高因子暴露的股票
- 做空低因子暴露的股票
- 目標：純因子收益，去除市場風險

AQR 實踐：
- 多因子模型（價值+動量+質量）
- 全球範圍
- 行業/國家中性
```

### 策略 2：风格溢价（Style Premia）

```
方法：
- 跨資產類別應用相同因子
- 股票、債券、外匯、商品
- 分散化因子暴露

例如動量：
- 股票：買漲的股票
- 債券：買漲的國債
- 外匯：買漲的貨幣
- 商品：買漲的商品

為什麼：同一個因子在不同資產都有效
```

### 策略 3：风险平价（Risk Parity）

```
方法：
- 按風險而非資本配置
- 每類資產貢獻相等風險
- 通常對債券加槓桿

與 Dalio 的異同：
- 相同：風險平衡理念
- 不同：AQR 更系統化、更學術化
```

## DECISION_RULES（决策规则）

### 因子选择规则
```
IF 因子有學術研究支持 AND 有經濟學邏輯 AND 跨市場有效
   THEN 可以納入
   BECAUSE 不是數據挖掘

IF 因子只在回測中有效 BUT 無法解釋為什麼
   THEN 謹慎
   BECAUSE 可能是過擬合

IF 因子近期表現很好 AND 變得擁擠
   THEN 減少暴露
   BECAUSE 擁擠會降低未來收益
```

### 组合构建规则
```
IF 單一因子暴露
   THEN 接受高波動
   BECAUSE 因子有艱難期

IF 多因子組合
   THEN 期望更平滑收益
   BECAUSE 因子間負相關

IF 因子之間相關性上升
   THEN 檢查市場環境
   BECAUSE 可能是極端時期
```

### 再平衡规则
```
IF 偏離目標權重超過閾值
   THEN 再平衡
   BECAUSE 維持因子暴露

IF 交易成本 > 再平衡收益
   THEN 不交易
   BECAUSE 成本控制

IF 因子信號極端強
   THEN 可以提前再平衡
   BECAUSE 抓住強信號
```

### 因子时机规则（谨慎使用）
```
IF 價值 spread 極度擴張（如 2020）
   THEN 增加價值暴露
   BECAUSE 長期均值回歸

IF 動量崩潰（如 2009.03）
   THEN 減少動量暴露
   BECAUSE 動量反轉風險

注意：因子擇時很難，Asness 自己也承認
「我們不太做因子擇時，但在極端情況會調整」
```

## 📈 因子表现特征

### 长期（1927-2023）

| 因子 | 年化超额收益 | 夏普比率 | 最大回撤 |
|------|-------------|---------|---------|
| 价值 | ~3-5% | ~0.3 | -50%+ |
| 动量 | ~6-8% | ~0.5 | -40%+ |
| 规模 | ~2-3% | ~0.2 | -30%+ |
| 质量 | ~3-4% | ~0.4 | -20%+ |
| **价值+动量** | ~5-7% | ~0.6 | -25% |

### 艰难时期

| 因子 | 艰难时期 | 最大落后 |
|------|---------|---------|
| 价值 | 1998-2000, 2017-2020 | 连续 3 年落后 50%+ |
| 动量 | 2009.03, 2020.11 | 单月 -30%+ |
| 规模 | 1980s, 2010s | 几乎消失 |
