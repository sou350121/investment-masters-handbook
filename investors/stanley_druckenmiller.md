# Stanley Druckenmiller 投资框架

## QUICK_FACTS
- **基金**：Duquesne Capital (已关闭) / Quantum Fund
- **规模**：峰值 $12B
- **核心风格**：流动性决定一切 + 集中重注 + 灵活反转
- **最佳适用**：宏观择时、流动性追踪、Fed 政策判断

> 重点：宏观择时、流动性追踪、集中下注；「保持灵活并在确定时下重注」。

## 核心原则
1. **宏观优先**：经济动能、央行政策、流动性是主线；个股服务于宏观方向。
2. **集中 & 重注**：看对方向时不怕重仓；小仓位意味「还没想清楚」。
3. **灵活反转**：错了快认错；盈利也能平仓；无沉没成本依恋。
4. **流动性至上**：「流动性推动市场，而非基本面」；关注 Fed 资产负债表、TGA、逆回购。
5. **技术面辅助**：用价格确认宏观判断；趋势、成交量、市场宽度是验证工具。

## 宏观判断框架
| 维度 | 观测指标 | 行动 |
|------|----------|------|
| 流动性 | Fed B/S、净流动性、M2 增速 | 上升 → 做多风险；下降 → 减仓或做空 |
| 经济周期 | ISM、就业、消费信心 | 上升期做多高β，衰退期偏防守 |
| 利率预期 | FF 期货、SOFR-OIS、实质利率 | 转鸽 → 先入股 → 等信用追赶 |
| 美元强弱 | DXY、新兴货币压力 | 强美元 → 新兴、商品承压 |

## 快速检查清单
- **方向性确信**：能否用一句话讲清逻辑？若需多重假设 → 减仓。
- **流动性趋势**：当前净流动性（Fed B/S − TGA − RRP）方向？3 个月变化？
- **市场宽度**：多少比例股票 > 200MA？若方向做多但宽度恶化 → 警惕。
- **止损/风控**：进场前设好退出点；错误时能否快速平仓？
- **机会成本**：这笔交易的预期报酬 vs. 现金或更好机会？

## 与宏观/流动性的连结
- **QT → QE 转换**：Druckenmiller 典型「闻到 Fed 转向」先布局成长股/风险资产。
- **高利率尾声**：通常在最后一次加息前后做多债券，等待降息周期。
- **美元趋势**：强美元时减少新兴、商品；弱美元时增加。
- **危机时刻**：流动性枯竭 → 可能出现超额机会（1992 英镑、2008 美股、2020 疫情）。

## DECISION_RULES（决策规则）

### 做多规则
```
IF 淨流動性（Fed B/S − TGA − RRP）上升 AND 估值合理
   THEN 做多風險資產
   BECAUSE 流動性推動市場

IF Fed 轉向（緊→鬆）信號確認
   THEN 先佈局成長股
   BECAUSE 流動性改善先反映在股市

IF 確信度 90%+ AND 邏輯一句話講清楚
   THEN 重倉 10-25%
   BECAUSE 看對方向要敢下注

IF 最後一次加息確認
   THEN 做多長期國債
   BECAUSE 歷史規律
```

### 做空/减仓规则
```
IF 淨流動性下降 AND 高估值
   THEN 減倉或做空
   BECAUSE 雙殺風險

IF 方向做多 BUT 市場寬度惡化（< 50% 股票 > 200MA）
   THEN 警惕，考慮減倉
   BECAUSE 市場內部轉弱

IF 需要多重假設才能支持邏輯
   THEN 減倉
   BECAUSE 還沒想清楚
```

### 认错规则
```
IF 論點被證偽 OR 價格確認反向
   THEN 立即止損
   BECAUSE 錯了快認錯

IF 虧損但論點仍成立 AND 流動性未變
   THEN 可以持有/加倉
   BECAUSE 時機問題而非方向問題
```

### 仓位规则
```
IF 確信度 90%+ THEN 重倉 10-25%
IF 確信度 70% THEN 中等 5-10%
IF 確信度 50% THEN 小倉位或不做
```
