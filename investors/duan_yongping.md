## 🧠 商业模式分析框架

### 什么是好公司？

```
好公司的特征：
1. 有差异化（别人做不了或做不好）
2. 有定价权（能提价或不降价）
3. 有复利效应（时间是朋友）
4. 管理层正直（不骗人）
```

### 商业模式的本质

| 类型 | 说明 | 例子 |
|------|------|------|
| **差异化** | 产品/服务有独特价值 | 苹果、茅台 |
| **成本领先** | 同样品质更低成本 | Costco |
| **网络效应** | 用户越多价值越大 | 微信、淘宝 |
| **品牌心智** | 占据用户心智 | 可口可乐、茅台 |

## 📊 投资检查清单

### 买入前问自己

```
□ 1. 这家公司我懂吗？（能力圈）
□ 2. 这是一个好生意吗？（商业模式）
□ 3. 管理层值得信任吗？（诚信）
□ 4. 现在价格合理吗？（估值）
□ 5. 我愿意持有10年吗？（长期）
```

### 不做什么（负面清单）

- ❌ 不做空（做空是借钱赌下跌，风险无限）
- ❌ 不借钱投资（杠杆会放大错误）
- ❌ 不投不懂的公司（能力圈外）
- ❌ 不频繁交易（摩擦成本）
- ❌ 不预测短期走势（没人能预测）

## 🏆 经典投资案例

### 案例一：网易 (2002)

```
背景：
- 互联网泡沫破裂，网易跌到 $0.8
- 面临退市风险，华尔街一片看空

段永平的分析：
- 游戏业务现金流很好
- 市值 < 手上现金
- 管理层（丁磊）正直、能干

结果：
- 买入价约 $1
- 持有至 $40+ 卖出
- 回报 40 倍+
```

### 案例二：苹果

```
投资逻辑：
- 生态系统（iOS + App Store）形成壁垒
- 用户粘性极高（换机成本高）
- 品牌溢价能力强
- 现金流极其充沛

核心观点：
"苹果不是硬件公司，是软件生态公司。"
```

### 案例三：茅台

```
投资逻辑：
- 品牌心智无可替代
- 定价权：越涨越买
- 产品不怕库存（越老越值钱）
- 几乎不需要再投资

核心观点：
"茅台的生意模式是印钞机级别的。"
```

## DECISION_RULES

### 买入规则

```
IF 能2句话说清商业模式 AND 有差异化/定价权
   THEN 在能力圈内
   BECAUSE 不懂不做 (段永平)

IF 好公司 AND 价格合理 AND 愿意持有10年
   THEN 考虑买入
   BECAUSE 买股票就是买公司 (段永平)

IF ROIC > 15% 持续多年 AND 不需大量再投资
   THEN 好生意
   BECAUSE 自由现金流是真金白银 (段永平)

IF 管理层诚信 AND 与股东利益一致 AND 能力强
   THEN 管理层可信
   BECAUSE 好生意也需要好管理 (段永平)

IF 价格 < 内在价值 × 0.7 AND 好公司
   THEN 好价格
   BECAUSE 安全边际 (段永平)
```

### 不买规则

```
IF 无法用2句话解释如何赚钱
   THEN 不买
   BECAUSE 不懂不做 (段永平)

IF 需要不断融资才能活
   THEN 不买
   BECAUSE 不是好生意 (段永平)

IF 管理层有诚信问题
   THEN 不买
   BECAUSE 骗过一次会骗第二次 (段永平)

IF 估值需要"信仰"支撑
   THEN 不买
   BECAUSE 价格太贵 (段永平)

IF 竞争激烈无差异化
   THEN 不买
   BECAUSE 没有护城河 (段永平)
```

### 卖出规则

```
IF 商业模式被破坏 OR 护城河消失
   THEN 卖出
   BECAUSE 好公司变坏了 (段永平)

IF 管理层出现诚信问题
   THEN 立即卖出
   BECAUSE 无法信任 (段永平)

IF 估值极度高估（如 PE > 50 且增速放缓）
   THEN 可以考虑卖出
   BECAUSE 价格太贵（但好公司可以多拿）(段永平)

IF 有更好的机会 AND 确定性更高
   THEN 可以换仓
   BECAUSE 机会成本 (段永平)
```

### 持有规则

```
IF 好公司 AND 估值合理 AND 长期逻辑未变
   THEN 继续持有
   BECAUSE 时间是好公司的朋友 (段永平)

IF 股价下跌 BUT 基本面未变
   THEN 持有或加仓
   BECAUSE 市场先生情绪化 (段永平)

IF 短期业绩波动 BUT 长期竞争力未变
   THEN 继续持有
   BECAUSE 不要因为短期波动卖好公司 (段永平)
```

### 仓位规则

```
IF 确定性高 AND 了解深入 AND 价格好
   THEN 可以重仓
   BECAUSE 机会来了要敢下手 (段永平)

IF 杠杆 OR 借钱
   THEN 不做
   BECAUSE 杠杆会让你在对的时候也被迫出局 (段永平)

IF 仓位让你睡不着觉
   THEN 减仓
   BECAUSE 投资应该让生活更好，不是更焦虑 (段永平)
```
