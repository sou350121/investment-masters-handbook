## 🧠 选股三把刀：品牌、渠道、成本

邱国鹭认为，真正的护城河只有三种：

### 🔪 第一把刀：品牌
- 消费者愿意为品牌支付溢价
- 品牌能穿越周期、抵御通胀
- **判断标准**：提价后销量是否下降？

### 🔪 第二把刀：渠道
- 渠道为王，得终端者得天下
- 下沉市场的渠道覆盖难以复制
- **判断标准**：竞争对手能否在3年内复制？

### 🔪 第三把刀：成本
- 低成本是最持久的竞争优势
- 规模效应、地理位置、技术优势
- **判断标准**：成本优势来自结构性因素还是暂时因素？

## 🏭 行业选择：得寡头者得天下

### 行业集中度理论

```
行业发展阶段：
初创期 → 成长期 → 洗牌期 → 成熟期 → 寡头垄断

最佳投资时点：洗牌期末期 → 寡头形成初期
```

### 行业选择标准

| 特征 | 好行业 | 差行业 |
|------|--------|--------|
| **集中度** | CR3 > 60% | 分散，无龙头 |
| **增速** | 稳定或温和增长 | 高增长（引来竞争） |
| **差异化** | 产品有差异 | 同质化严重 |
| **进入壁垒** | 高，需要时间积累 | 低，资本可复制 |
| **定价权** | 有，可提价 | 无，价格战 |

---

## 📊 估值锚：不要为成长付太多溢价

### 估值原则

1. **PE 锚**：好公司 15-20x PE 合理，>30x 需要非常高的确定性
2. **PB 锚**：ROE 15% 的公司，1.5-2x PB 合理
3. **历史区间**：参考公司/行业历史估值中枢
4. **相对估值**：同行业横向比较

### 便宜的定义

```
便宜 ≠ PE 低
便宜 = 价格 < 内在价值 × 0.7
```

## DECISION_RULES

```
IF 行业CR3 > 50% AND 龙头市占率持续提升
   THEN 行业值得关注
   BECAUSE 得寡头者得天下

IF 行业CR3 < 30% AND 玩家众多
   THEN 回避该行业
   BECAUSE 格局未定，价格战风险

IF 公司有品牌溢价能力（提价不掉量）
   THEN 护城河有效
   BECAUSE 品牌是真护城河

IF 公司有渠道优势 AND 竞争对手3年内无法复制
   THEN 护城河有效
   BECAUSE 渠道为王

IF 成本优势来自规模/地理/资源 AND 非暂时性
   THEN 护城河有效
   BECAUSE 结构性成本优势持久

IF PE > 30x AND 增速 < 30%
   THEN 估值偏贵
   BECAUSE PEG > 1 为成长付太多溢价

IF PE < 15x AND ROE > 15% AND 护城河完好
   THEN 好价格
   BECAUSE 便宜的好公司

IF 股价因短期利空下跌 > 30% AND 长期逻辑未变
   THEN 逆向机会
   BECAUSE 人弃我取

IF 市场极度悲观 AND 机构仓位处于历史低位
   THEN 布局时机
   BECAUSE 别人恐惧我贪婪

IF 估值处于历史高位 AND 市场情绪亢奋
   THEN 降低仓位
   BECAUSE 不和市场对着干太久

IF 公司基本面恶化（护城河受损）
   THEN 卖出
   BECAUSE 便宜可能更便宜

IF 找到更好的标的 AND 当前持仓性价比下降
   THEN 换仓
   BECAUSE 机会成本考量
```

## ✅ 买入检查清单

### 行业维度
- [ ] 行业集中度 CR3 > 50%？
- [ ] 行业有定价权（非价格战）？
- [ ] 行业增速适中（10-20%），不会引来太多竞争？
- [ ] 行业有进入壁垒？

### 公司维度
- [ ] 公司有"三把刀"之一（品牌/渠道/成本）？
- [ ] 护城河是真的还是伪护城河？
- [ ] 管理层诚信、能力如何？
- [ ] 公司 ROE > 15%？

### 估值维度
- [ ] PE 是否处于历史合理区间？
- [ ] 与同行相比是否便宜？
- [ ] PEG < 1？
- [ ] 有没有隐藏资产/负债？

### 时机维度
- [ ] 是否有短期利空导致的错杀？
- [ ] 机构持仓是否处于低位？
- [ ] 市场情绪是否悲观？
