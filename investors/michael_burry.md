# Michael Burry 投资框架

## QUICK_FACTS
- **基金**：Scion Asset Management（2013-至今）
- **规模**：~$250M
- **核心风格**：逆向深挖 + 极端集中 + 独立研究
- **最佳适用**：泡沫识别、逆向机会、做空策略

> **风格**：逆向深挖、集中持仓、不从众

## 核心原则
1. **独立研究**：不依赖卖方报告；自己挖 SEC 文件、数据、细节。
2. **极端逆向**：共识错误时下重注；接受长期被市场「打脸」。
3. **集中持仓**：找到高确信机会就重仓；分散是「无知的保护」。
4. **价值锚定**：用清算价值、重置成本等硬指标估值。
5. **长期视野**：愿意等待数年让论点兑现。

## 经典战役
| 年份 | 事件 | 逻辑 |
|------|------|------|
| 2005-2008 | 做空次贷（CDS） | 次级房贷违约率必然飙升 |
| 2019 | 做多 GameStop | 被过度做空、有资产价值 |
| 2021 | 警告通膨/泡沫 | 流动性过剩、估值极端 |
| 2022-23 | 做空 TSLA/AAPL | 估值过高、周期风险 |

## 研究方法
1. **读原始文件**：10-K、10-Q、信用评级报告、贷款池数据。
2. **找不一致**：管理层说法 vs 财务数据 vs 行业趋势。
3. **计算硬底**：清算价值、有形资产、现金流折现。
4. **测试共识**：为什么市场这样定价？共识可能错在哪？

## 快速检查清单
- **共识是什么？** 市场为何这样定价？
- **共识哪里错？** 有什么数据/逻辑被忽略？
- **硬底在哪？** 最坏情况的清算/残值？
- **时间框架？** 论点兑现需要多久？能撑住吗？
- **仓位规模？** 确信度与仓位匹配吗？

## 与宏观/周期的连结
- **信用泡沫末期**：寻找被忽视的结构性风险（如 2005 次贷）。
- **极端估值**：泡沫顶部做空高估值成长股。
- **通膨/利率转向**：提前布局受益于利率上升的标的。
- **流动性收紧**：高杠杆、负现金流公司最危险。

## DECISION_RULES（决策规则）

### 逆向买入规则
```
IF 共識明顯錯誤 AND 有數據/邏輯支持
   THEN 深入研究
   BECAUSE 共識錯誤是超額回報來源

IF 股價像「路殺」(road kill) AND 資產價值仍在
   THEN 考慮買入
   BECAUSE 極端低估

IF 清算價值 > 市值 AND 有催化劑
   THEN 考慮買入
   BECAUSE 硬底保護

IF 高確信 AND 論點獨立驗證
   THEN 重倉
   BECAUSE 集中在最好的機會
```

### 做空规则
```
IF 估值極端 AND 需要「信仰」合理化
   THEN 考慮做空
   BECAUSE 泡沫終會破

IF 結構性風險被市場忽視（如 2005 次貸）
   THEN 深入研究做空機會
   BECAUSE 共識盲點

IF 流動性收緊 AND 高槓桿/負現金流公司
   THEN 做空候選
   BECAUSE 最危險的標的
```

### 研究规则
```
IF 賣方報告說好
   THEN 獨立驗證
   BECAUSE 不依賴賣方

IF 管理層說法
   THEN 對照財務數據與行業趨勢
   BECAUSE 找不一致

IF 市場定價
   THEN 問「為什麼？共識哪裡錯？」
   BECAUSE 測試共識
```

### 风控规则
```
IF 論點兌現需要數年
   THEN 確保資金續航
   BECAUSE 可能長期被「打臉」

IF 做空
   THEN 嚴格控制倉位
   BECAUSE 無限虧損風險

IF 時機太早
   THEN 可以承受但要有預案
   BECAUSE 時機是最難的
```
