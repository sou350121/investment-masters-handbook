# Michael Burry 投资框架：原始数据与逆向博弈

> "我所有的交易都建立在数据的基础上。" —— 麦克·伯里

## 📋 伯里分析工具箱 (The Research Toolkit)

### 1. 原始文件挖掘 (SEC Filing Scavenger)
- 伯里几乎不看卖方报告。他的所有信息源来自：10-K（年报）、10-Q（季报）、8-K（重大事件）以及成千上万页的招股说明书和贷款支持证券（MBS）底层数据。
- 关键：寻找管理层描述与数字细节之间的不一致性。

### 2. 识别“路杀”资产 (Road-Kill Assets)
- 伯里寻找那些像“马路上被撞死的动物”一样、被市场唾弃的廉价公司。
- 估值锚点：清算价值、净营运资本（Net-Net）以及极低的 EV/EBITDA。

### 3. 做空的科学
- 寻找“由于虚假信仰驱动的不可持续估值”。
- 关键指标：极端的信贷杠杆、不可持续的现金流缺口、以及由于流动性泛滥产生的价格扭曲。

## 🚫 绝对禁止清单 (The NEVER List)
- **绝不随大流**：如果一个投资逻辑已经成为 Twitter 或 Reddit 上的热门，那就不是伯里的风格。
- **绝不忽视“细节中的恶魔”**：如果贷款合同里有一行小字可能导致违约，你必须把它找出来。
- **绝不在证明自己错误（数据证伪）后死守**：虽然伯里以坚持著称，但如果核心数据发生反转，他会果断清仓。
- **绝不使用复杂的预测模型来粉饰估值**：如果一家公司在现在的清算价值都不够，那它的未来假设就是废话。
- **绝不被管理层的个人魅力迷惑**：伯里只看数字，数字从不撒谎。

## 📖 经典案例研究 (Case Studies)

### 1. 2008 年做空次贷：Scion 的成名战
- **背景**：通过逐个检查数千份次级抵押贷款池，伯里发现违约率在 2005 年就已经开始上升，而房产价格仍在上涨。
- **逻辑**：创造性地要求银行设计针对次贷的信用违约互换（CDS）。
- **结果**：虽然等待了三年并承受了巨额保费支出，但最终在系统崩溃时获得了几百倍的回报。

### 2. GameStop (2019)：在废墟中发现价值
- **背景**：当时 GME 被视为垂死的实体店。
- **逻辑**：伯里发现其资产负债表上有极高的现金，且索尼和微软即将推出带光驱的新一代主机（这意味着实体碟不会立即消亡）。
- **结果**：他在 GME 暴涨前布局，并在其成为 Meme 股疯狂上涨前获利了结，体现了精准的价值锚定。

## 💡 决策规则 (DECISION_RULES)

### 独立研究与数据规则 (Research & Data)
```text
IF 卖方报告一致看好 AND 估值处于历史高位 AND 核心财务数据 (如经营现金流) 开始恶化
   THEN 忽略所有噪音，启动详细的空头调查
   BECAUSE 共识往往是滞后的，只有原始数据能反映真相

IF 某公司的 EV/EBITDA < 3 AND 业务在可见的 3 年内不会破产
   THEN 视为潜在的“路杀”资产进入候选名单
   BECAUSE 这种极端的低估通常预示着巨大的修复潜力

IF 发现贷款合同或招股说明书中存在显著的法律或结构性风险
   THEN 赋予该资产极高的风险溢价，甚至直接排除
   BECAUSE 细节决定了在极端情况下的生存能力
```

### 泡沫与空头规则 (Bubble & Short)
```text
IF 市场出现“被动投资 (Index Fund) 泡沫” (资金无差别流入指数成分股，导致估值错位)
   THEN 寻找那些被指数排除但基本面卓越的小型股
   BECAUSE 指数基金的机械性买入创造了结构性的低估区

IF 做空某个标的 AND 该公司开始进行大规模的股份回购或寻找到了新的长期融资
   THEN 重新评估空头逻辑，必要时立即止损
   BECAUSE 充足的流动性可以延后泡沫的破裂时间，甚至改变博弈逻辑

IF 在泡沫顶点做空
   THEN 必须控制仓位规模 (Size) 且做好“太早” 2-3 年的心理准备
   BECAUSE 市场保持非理性的时间往往比你的头寸能支撑的时间长
```
