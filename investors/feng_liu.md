## 🎯 核心投资哲学：弱者体系

### 什么是弱者体系？

> "我把自己定位成弱者——信息弱势、资源弱势、能力弱势。既然是弱者，就要假设市场比我聪明，股价已经反映了大部分已知信息。"

### 弱者 vs 强者

| 维度 | 强者体系 | 弱者体系（冯柳） |
|------|----------|------------------|
| **信息假设** | 我比市场知道更多 | 市场比我知道更多 |
| **预测能力** | 相信自己能预测 | 承认无法预测 |
| **买入时机** | 右侧确认后买入 | 左侧逆向买入 |
| **止损策略** | 设止损 | 基本不止损 |
| **择时** | 相信能择时 | 放弃择时 |
| **持仓风格** | 集中 + 频繁调整 | 分散 + 长期持有 |

## 🧠 核心原则

### 一、假设市场是对的

> "如果股价跌了50%，不要先想'市场错了'，而是想'市场为什么对了'。只有想清楚市场对的逻辑，才能判断它是否可能错。"

**实践方法**：
1. 股价下跌时，先列出市场担心的所有理由
2. 逐一分析这些担心是否会成为现实
3. 如果担心被过度放大，才是机会

### 二、赔率优先于胜率

> "我追求的是赔率，不是胜率。宁愿做10次亏5次但每次赚3倍的事，也不愿做10次赢9次但每次只赚10%的事。"

**计算方法**：
```
期望收益 = 胜率 × 盈利幅度 - 败率 × 亏损幅度

例：
方案A：80%胜率，赚20%；20%败率，亏10%
期望 = 0.8 × 20% - 0.2 × 10% = 14%

方案B：40%胜率，赚100%；60%败率，亏30%
期望 = 0.4 × 100% - 0.6 × 30% = 22%

选 B（赔率优先）
```

### 三、逆向左侧买入

> "好价格只有在别人不看好的时候才会出现。如果等到所有人都看好，股价早就涨上去了。"

**左侧买入的条件**：
1. 股价已经大幅下跌（通常 >30%）
2. 市场担心的因素是可逆的或被夸大的
3. 公司长期价值没有根本性损害
4. 有足够的"忍受时间"

### 四、不择时、不止损

> "择时意味着你认为自己能预测市场，止损意味着你承认自己买错了。我既不相信自己能预测市场，也尽量让自己不买错。"

**不止损的前提**：
- 买入前做足功课，确保长期逻辑正确
- 买入价格有足够安全边际
- 能承受账面亏损的心理压力
- 不用杠杆

## 📊 选股框架

### 寻找"可能错"的机会

```
市场共识 → 分析共识是否正确 → 寻找共识可能错的点

三种机会：
1. 市场过度悲观（利空被夸大）
2. 市场忽视正在发生的变化（慢变量）
3. 市场对长期价值定价不足（久期错配）
```

### 好股票的特征

| 特征 | 说明 |
|------|------|
| **估值便宜** | PE/PB 处于历史低位或相对低位 |
| **预期很低** | 市场已经 price in 坏消息 |
| **边际改善** | 有边际好转的迹象（但市场还没反应） |
| **向下有底** | 清算价值/重置成本提供保护 |
| **向上有空间** | 如果担心消除，估值修复空间大 |

## DECISION_RULES

```
IF 股价下跌 > 30% AND 市场担心因素可量化
   THEN 分析担心是否被夸大
   BECAUSE 弱者体系：先假设市场是对的

IF 市场担心因素是周期性/一次性 AND 长期逻辑未变
   THEN 逆向机会
   BECAUSE 市场可能过度反应短期利空

IF 市场担心因素是结构性/不可逆
   THEN PASS
   BECAUSE 市场可能是对的

IF 赔率 > 3:1 AND 胜率 > 30%
   THEN 值得下注
   BECAUSE 赔率优先于胜率

IF 赔率 < 2:1 即使胜率 > 70%
   THEN 谨慎
   BECAUSE 期望收益可能不够

IF 估值处于历史底部区间 AND 基本面有边际改善迹象
   THEN 左侧买入
   BECAUSE 价格已反映悲观预期

IF 股价新低 AND 找不到合理解释
   THEN 不要抄底
   BECAUSE 可能有我不知道的利空

IF 买入后继续下跌 AND 长期逻辑未变
   THEN 持有或加仓
   BECAUSE 不止损是弱者体系核心

IF 买入逻辑被证伪（长期逻辑改变）
   THEN 卖出
   BECAUSE 认错要快

IF 股价涨到合理估值 AND 没有更多上涨空间
   THEN 逐步减仓
   BECAUSE 赔率变差

IF 有更好的赔率机会
   THEN 换仓
   BECAUSE 机会成本

IF 市场极度恐慌 AND 好公司被无差别抛售
   THEN 买入优质股
   BECAUSE 最好的机会出现在最差的时刻

IF 使用杠杆
   THEN 不适用弱者体系
   BECAUSE 弱者体系需要"熬"的能力
```

## ✅ 买入检查清单

### 市场定价分析
- [ ] 市场担心什么？列出 3 个以上理由
- [ ] 每个担心的概率和影响有多大？
- [ ] 担心是被夸大了还是被低估了？
- [ ] 如果担心成真，最坏情况是什么？

### 赔率评估
- [ ] 向下空间有多大？（亏损幅度）
- [ ] 向上空间有多大？（盈利幅度）
- [ ] 赔率是否 > 3:1？
- [ ] 胜率估计是多少？

### 时间维度
- [ ] 能忍受多长时间的账面亏损？
- [ ] 催化剂可能什么时候出现？
- [ ] 有没有不能等的理由？

### 风险控制
- [ ] 是否使用杠杆？（不应该）
- [ ] 单只股票仓位是否过重？
- [ ] 有没有相关性风险？

## 🔄 卖出框架

### 应该卖出的情况

1. **逻辑证伪**：长期投资逻辑被推翻
2. **估值到位**：股价涨到合理估值或高估
3. **更好机会**：发现赔率更好的标的
4. **基本面恶化**：公司质地变差

### 不应该卖出的情况

1. **短期下跌**：股价波动不是卖出理由
2. **市场恐慌**：别人恐惧时不是卖出时机
3. **账面亏损**：弱者体系不止损

## ⚠️ 适用场景与局限

### ✅ 最适合
- A 股逆向投资

- 市场恐慌时的买入决策

### ⚠️ 局限
- 需要很强的心理承受能力
- 不适合需要短期业绩的资金
- 在趋势行情中可能跑输
- 对选股能力要求很高（因为不止损）
