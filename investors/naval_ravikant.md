# Naval Ravikant 投资与财富哲学

> "财富是在你睡觉时也能为你赚钱的资产。金钱是转账的记账符号。地位是你在社会阶层中的位置。" —— 纳瓦尔·拉维康特

## 📋 纳瓦尔的“财富一页纸”

### 1. 建立“特殊知识” (Specific Knowledge)
- 找到那些对你来说像是在玩，对别人来说却像是工作的领域。
- 特殊知识不能被教会，但可以被学习。它是你天赋、兴趣和好奇心的交集。

### 2. 运用杠杆 (Leverage)
- **劳动力**：最古老、最难管理的杠杆（需要别人追随你）。
- **资本**：更现代的杠杆（需要别人给你钱）。
- **代码与媒体**（无须许可的杠杆）：这是新时代的杠杆。你可以编写一段代码，或者制作一个视频，它们在你睡觉时也为你工作。

### 3. 判断力高于努力 (Judgment over Effort)
- 在现代自动化时代，一个正确的决定可以带来数千倍的收益，而努力工作的边际收益正在递减。

---

## 🚫 纳瓦尔的“避雷清单”
- **绝不参与“地位游戏”**：地位是一个零和博弈（为了让你赢，别人必须输）。财富是正和博弈。
- **绝不与那些“悲观主义者”合作**：悲观在短期内听起来很聪明，但乐观在长期内才能赚钱。
- **绝不为了短期的金钱牺牲长期的复利**：无论是人际关系还是财富，所有的收益都来自于复利。
- **绝不接受任何“计分式工作”**：按小时计费的工作永远无法让你致富，你必须拥有资产（股权）才能获得财务自由。
- **绝不忽视自己的“平静”**：如果一个赚钱机会让你失去睡眠，那它就是一笔亏损。

---

## 📖 核心心智模型 (Mental Models)

### 1. 概率思维 (Probabilistic Thinking)
- 投资是胜率与赔率的博弈。纳瓦尔倾向于押注那些即使失败了损失也有限，但一旦成功就能带来 100 倍收益的“不对称机会”。

### 2. 委托代理问题 (Principal-Agent Problem)
- 最好的创始人是那些把自己当成“委托人”而不是“代理人”的人。如果一个创始人只关心自己的工资或退出，那就不要投资他。

---

## 💡 决策规则 (DECISION_RULES)

### 选人与选赛道规则 (Founder & Sector)
```text
IF 创始人具备“特殊知识” AND 能够利用“代码/媒体”建立零许可杠杆
   THEN 极力买入/投资，并长期持有
   BECAUSE 这种组合在现代经济中具有极高的缩放潜力和极低的边际成本

IF 一个项目依赖于大量的低廉劳动力或受监管的资本杠杆
   THEN 保持警惕或避开
   BECAUSE 这种杠杆具有高度的脆弱性，且难以在不增加管理复杂性的情况下进行扩张

IF 合作伙伴表现出短视行为或在微小利益上不诚实
   THEN 立即终止合作，不再进行任何后续投入
   BECAUSE 所有显著的收益都来自长期复利，而复利的前提是信任的无损传递
```

### 个人投资与退出规则 (Personal Strategy)
```text
IF 某项资产的上涨空间是有限的 (如租金、工资) AND 投入的精力是持续的
   THEN 寻找机会将其置换为具有无限上涨潜力的“期权型资产”
   BECAUSE 真正的致富需要脱离时间与报酬的线性关系

IF 市场陷入狂热 AND 你发现自己开始关心社交媒体上的排名或地位
   THEN 强制离场或减少关注
   BECAUSE 地位游戏会扭曲你的判断力，导致你在估值顶部追逐群体共识

IF 某项投资需要你频繁查看报价以获得心理安宁
   THEN 卖出直到你可以安睡为止
   BECAUSE 平静是最高级的财富，焦虑说明你并不真正理解或拥有该项资产
```
