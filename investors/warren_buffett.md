# Warren Buffett 投资框架

## QUICK_FACTS
- **基金**：Berkshire Hathaway
- **规模**：$900B+
- **核心风格**：护城河 + 安全边际 + 长期持有
- **最佳适用**：选股估值、护城河评估、资本配置

> 聚焦「企业为债、股东为债权人」的现金流视角；强调护城河、资本配置与估值安全边际。

## 核心原则
- 企业如同债券：看「现金流票息」与「成长率」，折现决定价值。
- 护城河与竞争优势：品牌/网路效应/成本优势/转换成本。
- 资本配置纪律：再投资回报率 > 要求报酬率；若否，回购/分红。
- 杠杆谨慎：避免永久性损失风险；保留充足现金对衰退/捡便宜。
- 估值安全边际：好公司也需合理价；偏好可预测现金流、抗通膨能力。

## 快速检查清单
- 竞争力：市占/毛利稳定；ROIC 高于 WACC 且持续。
- 现金流质量：FCF/净利比、营运现金流占利润、存货/应收周转。
- 资本配置：再投资回报率、回购是否在合理价、负债期限/成本。
- 估值：收益率 vs 长端利率（或自定贴现率）；成长假设是否保守。
- 风险：关键依赖（单一产品/供应/监管）；周期敏感度。

## 与宏观的连结
- 实质利率上行 → 估值折现率上升，安全边际要求更高；偏好高现金转化、低杠杆。
- 流动性收缩/衰退期：持现金等待捡便宜；宁愿错过、不冒永久损失。
- 通膨环境：看定价权、成本转嫁力；寿命长的品牌/网路型护城河较抗侵蚀。

## 如何判断低估（Buffett 视角）

### 🎯 内在价值估算框架

**1. 现金流折现（Owner Earnings）**
```
內在價值 = Σ [未來自由現金流 ÷ (1 + 貼現率)^t]

Owner Earnings = 淨利
                + 折舊/攤銷
                - 維持性資本支出
                - 營運資本增加
```

**关键问题**：
- ✅ 未来 10 年现金流可预测吗？（护城河够宽？）
- ✅ 维持性资本支出是多少？（警惕低报的公司）
- ✅ 营运资本是否持续膨胀？（存货、应收堆积）
- ✅ 贴现率用多少？（10Y 国债 + 4-6% 风险溢价）

**2. 盈余收益率法（简化版）**
```
盈餘收益率 = Owner Earnings ÷ 市值
對比：10Y 國債收益率

若 盈餘收益率 > 國債收益率 × 1.5 → 可能低估
若 盈餘收益率 < 國債收益率 → 可能高估
```

**3. 相对估值（谨慎使用）**
- P/E vs 历史区间：但**只在护城河未变时有效**
- P/B 适用于金融/重资产：ROE 稳定时可用
- EV/EBITDA：适合比较资本结构不同的公司

**Buffett 特别提醒**：
> "估值不是科学，是艺术。但艺术也需要纪律。"
- 不要为了买而降低标准
- 宁可以合理价买优秀公司，也不以便宜价买平庸公司

### 🔍 低估的信号（逐项检查）

| 指标 | 低估信号 | 数字标准 | 陷阱警示 |
|------|----------|----------|----------|
| **盈余收益率** | > 10Y 国债 × 1.5 | 假设国债 4% → 需 > 6% | 盈余品质？一次性项目？ |
| **FCF Yield** | > 8-10% | FCF/市值 > 8% | 维持性支出被低估？ |
| **P/E** | < 历史均值 - 1σ | 需确认护城河未变 | 盈利见顶？周期高点？ |
| **P/B** | < 1.2 且 ROE > 15% | 金融股/重资产适用 | 资产减值？不良资产？ |
| **PEG** | < 0.8 | P/E ÷ 增速 | 增速不可持续？ |
| **EV/EBIT** | < 10 | 好生意：8-12 | 杠杆过高掩盖？ |
| **股息收益率** | > 国债 + 2% | 假设国债 4% → 需 > 6% | 股息不可持续？ |

---

### 💎 低估的定性信号

**市场情绪面**
- ✅ 该行业/公司被「暂时」冷落（非结构性衰退）
- ✅ 短期负面新闻（诉讼、召回、季报不佳）但长期不受影响
- ✅ 「无聊」的生意：没有成长故事，分析师不关注
- ✅ 指数调整/被动基金被迫卖出（非基本面原因）

**Buffett 经典案例**
| 公司 | 年份 | 低估原因 | 买入逻辑 |
|------|------|----------|----------|
| American Express | 1963 | 色拉油丑闻，股价腰斩 | 品牌护城河未损，暂时危机 |
| 华盛顿邮报 | 1973 | 市场崩盘，媒体被错杀 | 区域垄断地位，市值 < 清算价 |
| 富国银行 | 1990 | 房地产危机恐慌 | 管理优秀，危机中的错杀 |
| 可口可乐 | 1988 | 市场认为成长停滞 | 全球品牌价值被低估 |

**低估的必要条件（缺一不可）**
1. 护城河依然存在（竞争优势未被破坏）
2. 管理层诚信且有能力
3. 财务健康（低杠杆、现金流正）
4. 有安全边际（市价 < 内在价值的 60-70%）

## 如何判断高估（Buffett 视角）

### 🚨 高估的定量信号

| 指标 | 高估信号 | 危险水平 | Buffett 视角 |
|------|----------|----------|--------------|
| **盈余收益率** | < 10Y 国债收益率 | < 4%（国债 4% 时） | "你在为增长付溢价，但增长确定吗？" |
| **FCF Yield** | < 3% | 需 50+ 年回本 | "这是投资还是投机？" |
| **P/E** | > 历史均值 + 2σ | 科技股 > 40，传统 > 25 | "除非增速显著加快" |
| **P/B** | > 5 且 ROE < 20% | 支付溢价但回报平庸 | "为什么不买国债？" |
| **PEG** | > 2 | P/E / 增速 > 2 | "为成长付了太多" |
| **市值/GDP** | > 150% | Buffett 指标（美股） | "市场整体泡沫" |

### 🔥 高估的市场情绪信号

**狂热迹象（Buffett 多次提醒）**
- ✅ "这次不一样"的论调盛行
- ✅ 估值用「新指标」合理化（如用户数、点击率代替盈利）
- ✅ IPO 首日暴涨成常态，散户疯抢新股
- ✅ 杠杆融资创新高（保证金债务/GDP）
- ✅ 「股神已老」、「价值投资已死」成为共识
- ✅ 出租车司机/理发师都在谈论股票

**Buffett 经典语录**
> "只有退潮时，才知道谁在裸泳。"
> "别人贪婪时恐惧，别人恐惧时贪婪。"
> "当市场狂热时，我选择去打桥牌。"

### 📊 高估的行业/公司特征

**警示信号**
- [ ] **估值需要「信仰」**：需相信颠覆性技术、指数级增长
- [ ] **盈利遥遥无期**：「规模化后就盈利」的承诺
- [ ] **会计激进**：确认收入过早、资本化过多、一次性调整频繁
- [ ] **股权稀释严重**：员工期权、可转债不断稀释股东
- [ ] **自由现金流为负**：持续烧钱、依赖融资
- [ ] **并购狂魔**：靠并购做大营收、掩盖内生增长乏力
- [ ] **管理层高调**：频繁路演、承诺宏大愿景、股票质押

**Buffett 的谨慎**
| 情境 | Buffett 做法 | 为什么 |
|------|-------------|--------|
| 科技股泡沫（1999-2000） | 持有现金、不参与 | "不懂就不碰" |
| 加密货币热潮（2017-2021） | 公开批评「无价值」 | "不产生现金流" |
| SPAC 热潮（2020-2021） | 远离 | "结构性激励错位" |

### 🛡️ 避免高估的决策纪律

**1. 机会成本思维**
```
每個決策都在和「持有現金」或「國債」競爭

問自己：
- 這筆投資的預期回報 > 10-12%（長期股市均值）嗎？
- 風險調整後，真的比國債/現金好嗎？
- 如果用同樣的錢等 2 年，會有更好的機會嗎？
```

**2. 反向压力测试**
- 假设盈利增速减半，估值还合理吗？
- 假设利率上升 2%，贴现率对估值的影响？
- 假设护城河被侵蚀 20%，市价是否还有支撑？

**3. Buffett 的「打孔卡」理念**
> "假设你一生只有 20 个投资决策的机会（20 个打孔位），你还会买这个吗？"

**4. 等待「肥球」（Fat Pitch）**
- 不是每个球都要挥棒
- 等待「护城河强 + 管理层优秀 + 估值便宜」三者合一的机会
- 宁可错过，不要犯错

## DECISION_RULES（决策规则）

### 买入规则
```
IF 盈餘收益率 > 10Y國債 × 1.5 AND 護城河完好
   THEN 考慮買入
   BECAUSE 提供安全邊際，回報優於無風險利率

IF FCF Yield > 8% AND ROIC > 15% 持續 5 年+
   THEN 深入研究
   BECAUSE 可能是複利機器

IF 短期負面新聞 AND 護城河未損 AND 股價下跌 > 30%
   THEN 機會窗口
   BECAUSE 市場過度反應暫時性問題

IF P/E < 歷史均值 - 1σ AND 護城河未變 AND 管理層誠信
   THEN 考慮買入
   BECAUSE 週期性低估

IF 「無聊」的生意 AND 分析師不關注 AND FCF 穩定
   THEN 深入研究
   BECAUSE 可能是隱藏的好公司
```

### 卖出规则
```
IF 護城河被侵蝕 AND 無改善跡象
   THEN 賣出
   BECAUSE 競爭優勢消失

IF 估值 > 內在價值 × 1.5
   THEN 考慮減倉
   BECAUSE 安全邊際消失

IF 管理層誠信問題
   THEN 立即賣出
   BECAUSE 無法信任

IF 有更好的機會 AND 當前持倉估值合理但不便宜
   THEN 換倉
   BECAUSE 機會成本
```

### 不买规则
```
IF 無法用 2 句話解釋商業模式
   THEN 不買
   BECAUSE 能力圈外

IF 需要「信仰」未來才能合理化估值
   THEN 不買
   BECAUSE 投機而非投資

IF FCF 長期為負 AND 無明確盈利路徑
   THEN 不買
   BECAUSE 燒錢模式

IF 高槓桿 AND 短期債務 > 現金
   THEN 不買
   BECAUSE 衰退時破產風險

IF 管理層頻繁質押股票 OR 高調承諾
   THEN 不買
   BECAUSE 激勵錯位信號
```

## 估值小工具（快速计算）

### 工具 1：Owner Earnings 快算
```python
# 步驟 1：拿到財報數據
淨利 = 100
折舊攤銷 = 20
維持性資本支出 = 15  # 經驗：約折舊的 70-100%
營運資本增加 = 5

# 步驟 2：計算
Owner_Earnings = 淨利 + 折舊攤銷 - 維持性資本支出 - 營運資本增加
                = 100 + 20 - 15 - 5 = 100

# 步驟 3：估值
市值 = 1000
盈餘收益率 = 100 / 1000 = 10%

# 步驟 4：對比
國債收益率 = 4%
要求收益率 = 6%（國債 × 1.5）

結論：10% > 6%，可能低估（需進一步確認護城河）
```

### 工具 2：两阶段 DCF（保守假设）
```
假設條件：
- 前 5 年增速：8%
- 永續增速：3%（接近 GDP）
- 貼現率：10%（國債 4% + 6% 風險溢價）

當前 FCF = 100
市值 = 1000

計算：
階段 1（前 5 年）折現值 = 100 × [(1+8%)/(1+10%) + ... ] ≈ 416
階段 2（永續）折現值 = 100 × (1+8%)^5 × (1+3%)/(10%-3%) / (1+10%)^5 ≈ 872

內在價值 = 416 + 872 = 1288
安全邊際 = (1288 - 1000) / 1288 = 22%

結論：安全邊際 > 20%，可考慮（Buffett 偏好 > 30%）
```
