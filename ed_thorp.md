## 🎯 核心原则

### 原则 1：数学优先，情绪靠边

```
「市場是一個數學問題，不是心理遊戲。」

- 找到可量化的優勢
- 用歷史數據驗證
- 消除情緒干擾
- 機械化執行
```

### 原则 2：凯利公式——最优仓位管理

```
Kelly % = (bp - q) / b

其中：
- b = 賠率（贏時賺多少）
- p = 勝率
- q = 1 - p（敗率）

含義：
- 優勢越大，下注越多
- 沒有優勢，不下注
- 永遠不要超過凱利比例（避免破產）
```

**实际应用**：
```
假設：
- 勝率 55%（p = 0.55）
- 賠率 1:1（b = 1）
- q = 0.45

Kelly % = (1 × 0.55 - 0.45) / 1 = 10%

→ 每次下注 10% 資金
→ 實踐中用「半凱利」（5%）更安全
```

### 原则 3：套利优于投机

```
「我不預測市場方向，我找定價錯誤。」

套利的優點：
- 低風險或無風險
- 不依賴市場方向
- 可量化優勢
- 可重複執行
```

### 原则 4：风险管理是第一位

```
「避免破產，你就能活著等到優勢兌現。」

- 永遠不要全押
- 分散不相關的策略
- 設定最大回撤限制
- 保持流動性
```

## 📊 Thorp 的经典策略

### 策略 1：可转债套利（Convertible Arbitrage）

```
核心邏輯：
- 買入被低估的可轉債
- 做空對應股票對沖
- 賺取定價錯誤的收益

優勢：
- 下跌時有債券保護
- 上漲時有轉換權收益
- 波動率被低估時超額收益

Thorp 的創新：
在 Black-Scholes 之前，他就用自己的公式
定價可轉債中的期權價值
```

### 策略 2：权证套利（Warrant Arbitrage）

```
核心邏輯：
- 找到被錯誤定價的權證
- 用股票對沖 Delta
- 賺取時間價值衰減或定價修正

這是 Thorp 的起家策略，
在《Beat the Market》中詳細描述。
```

### 策略 3：统计套利（Statistical Arbitrage）

```
核心邏輯：
- 找到歷史上高度相關的股票對
- 當價差偏離均值時交易
- 等待價差回歸

要素：
- 配對選擇（同行業、相似業務）
- 入場閾值（偏離多少標準差）
- 出場規則（回歸到什麼程度）
- 止損設定（偏離繼續擴大怎麼辦）
```

## 🎲 凯利公式深度解析

### 为什么凯利公式重要？

```
問題：你有一個 60% 勝率、1:1 賠率的策略，
      每次該下注多少？

錯誤答案：
- 每次全押 → 早晚破產
- 每次固定金額 → 增長太慢
- 憑感覺 → 不穩定

正確答案：凱利公式 → 最優長期增長率
```

### 凯利公式的推导直觉

```
目標：最大化長期資本增長率

如果下注太少：
- 優勢沒有充分利用
- 增長太慢

如果下注太多：
- 波動太大
- 可能破產（一旦破產就歸零）

凱利公式找到了這個平衡點。
```

### 实战中的凯利调整

| 情境 | 调整 |
|------|------|
| 胜率/赔率估计不准 | 用半凯利或更低 |
| 波动率高 | 减少仓位 |
| 流动性差 | 减少仓位 |
| 相关性风险 | 分散后再计算 |
| 心理承受力低 | 用 1/4 凯利 |

### Thorp 的实际做法

```
「我通常用半凱利或更低。」

原因：
1. 估計的勝率/賠率可能有誤差
2. 黑天鵝事件
3. 心理舒適度
4. 保守比激進更能活下來
```

## 🛡️ 风险管理框架

### Thorp 的风险原则

```
1. 永遠不要讓單一事件毀掉你
2. 分散不相關的策略
3. 了解你的最大回撤
4. 保持足夠現金應對贖回
5. 槓桿要在控制範圍內
```

### 回撤控制

```
Princeton Newport Partners 的規則：
- 單月最大虧損限制
- 如果達到限制，減少倉位
- 等待波動率下降再恢復

結果：19 年只有 3 個月虧損
```

### 杠杆使用原则

```
「槓桿是雙刃劍，但如果你有真正的優勢，適度槓桿是合理的。」

Thorp 的做法：
- 槓桿比例：通常 2-4 倍
- 只在有明確優勢時使用
- 設定最大槓桿上限
- 壓力測試極端情況
```

## DECISION_RULES（决策规则）

### 策略评估规则
```
IF 勝率可量化 AND 賠率可估算 AND 歷史數據充足
   THEN 可以考慮
   BECAUSE 能計算期望值和凱利比例

IF 期望值 > 0 AND 夏普比率 > 1
   THEN 策略可行
   BECAUSE 正期望 + 風險調整後收益合理

IF 無法量化優勢
   THEN 不做
   BECAUSE 沒有數學基礎的交易是賭博
```

### 仓位管理规则
```
IF 有明確優勢 AND 勝率/賠率已知
   THEN 用半凱利計算倉位
   BECAUSE 保守比激進更能活下來

IF 估計不確定性高
   THEN 用 1/4 凱利或更低
   BECAUSE 估計誤差會放大風險

IF 策略之間有相關性
   THEN 總倉位要調低
   BECAUSE 相關性在極端時會上升
```

### 风控规则
```
IF 單月虧損達到限制（如 5%）
   THEN 減少倉位 50%
   BECAUSE 可能是模型失效或市場異常

IF 回撤超過歷史最大
   THEN 暫停交易，檢查模型
   BECAUSE 可能有未知風險

IF 流動性下降
   THEN 減少倉位
   BECAUSE 無法按計劃退出
```
