# Seth Klarman 投资框架

## QUICK_FACTS
- **基金**：Baupost Group（1982-至今）
- **规模**：$27B
- **核心风格**：深度价值 + 极端耐心 + 高现金比例
- **最佳适用**：特殊情况、困境债、冷门资产

> **风格**：深度价值、极端耐心、高现金比例

## 核心原则
1. **安全边际至上**：只在价格远低于保守估值时买入；宁可错过也不冒险。
2. **绝对回报思维**：不与指数比较；目标是「不亏钱」，而非「跑赢大盘」。
3. **耐心是武器**：可以持有大量现金数年等待机会；不怕跑输。
4. **冷门资产**：偏好流动性差、被忽视、结构性卖压的标的。
5. **风险 ≠ 波动**：真正风险是永久亏损；波动是创造机会的朋友。

## 投资机会分类
| 类型 | 特征 | 例子 |
|------|------|------|
| 困境债 | 公司困境但资产价值仍在 | 破产重组债券 |
| 清算/拆分 | 市场忽视拆分后价值 | Spin-off、清算股 |
| 被迫卖压 | 机构因规则被迫抛售 | 评级下调、指数剔除 |
| 冷门地区/行业 | 无人关注的市场 | 新兴市场、小型股 |

## 快速检查清单
- **内在价值？** 用保守假设算出的价值是多少？
- **安全边际？** 当前价格 vs 保守价值折扣多少？（目标 30-50%）
- **为何便宜？** 是结构性卖压还是基本面恶化？
- **下档保护？** 最坏情况能回收多少？
- **催化剂？** 什么事件会让市场重新定价？
- **流动性？** 能否承受长期持有、无法退出？

## 与宏观/周期的连结
- **恐慌/危机时期**：Baupost 最活跃的时候；现金储备是弹药。
- **低利率环境**：机会变少，现金比例提高；不追收益率。
- **信用周期尾声**：困境债机会增加；但需等违约潮真正到来。
- **流动性枯竭**：非流动资产暴跌 → 超额安全边际出现。

## 现金管理哲学
- 长期现金比例常达 30-50%
- 现金是「等待机会的期权」
- 不会为了「部署资金」而降低标准

## DECISION_RULES（决策规则）

### 买入规则
```
IF 安全邊際 > 30-50%（價格 vs 保守估值）
   THEN 考慮買入
   BECAUSE 足夠的下檔保護

IF 便宜因為結構性賣壓（指數剔除、評級下調）AND 基本面未變
   THEN 深入研究
   BECAUSE 被迫賣壓創造機會

IF 困境債 AND 資產價值 > 債券價格 AND 有催化劑
   THEN 深入研究
   BECAUSE 特殊情況機會

IF 恐慌/危機時期 AND 有現金儲備
   THEN 積極尋找機會
   BECAUSE Baupost 最活躍的時候

IF 冷門資產 AND 無人關注 AND 有價值
   THEN 深入研究
   BECAUSE 流動性折價
```

### 现金管理规则
```
IF 估值普遍高估 AND 缺乏安全邊際
   THEN 提高現金比例（可達 30-50%）
   BECAUSE 等待機會的期權

IF 機會不夠好
   THEN 不部署
   BECAUSE 不為部署而降低標準

IF 危機到來 AND 有現金
   THEN 大膽部署
   BECAUSE 現金是彈藥
```

### 不买规则
```
IF 安全邊際 < 30%
   THEN 不買
   BECAUSE 保護不足

IF 便宜因為基本面惡化 AND 無改善跡象
   THEN 不買
   BECAUSE 價值陷阱

IF 下檔保護不明確
   THEN 不買
   BECAUSE 最壞情況不可接受

IF 流動性極差 AND 無法長期持有
   THEN 不買
   BECAUSE 無法承受鎖定
```

### 卖出规则
```
IF 達到內在價值
   THEN 賣出
   BECAUSE 安全邊際消失

IF 有催化劑讓市場重新定價
   THEN 等待/部分獲利
   BECAUSE 價值實現

IF 基本面惡化
   THEN 重新評估，可能賣出
   BECAUSE 論點改變
```
