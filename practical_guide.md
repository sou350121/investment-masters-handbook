## 二、特殊策略實戰指南

### 🎯 策略 1：併購套利（Merger Arbitrage）

**適用場景**：已宣布的併購交易，標的價格 < 收購價

**Klarman 框架檢查**
- [ ] 交易類型：現金收購（風險低）vs 換股（受收購方股價影響）
- [ ] 監管風險：反壟斷審查概率？
- [ ] 融資風險：收購方有錢嗎？
- [ ] 股東批准：大股東態度？
- [ ] 競價可能：有其他買家嗎？
- [ ] 年化收益 vs 風險：價差÷預期完成時間 > 10%？

**操作要點**
```
買入標的（被收購方）
可選：做空收購方（若換股交易）
止損：交易破局宣布時立即退出
```

**案例參考**：微軟收購動視暴雪（2022-2023）、推特私有化（2022）

---

### 🎯 策略 2：困境債投資（Distressed Debt）

**適用場景**：公司面臨財務困境但有重組價值

**Klarman + Marks 框架**
- [ ] 清算價值 > 債券價格？
- [ ] 資本結構：你在哪一層？優先順序？
- [ ] 回收率估算（保守假設）
- [ ] 管理層/法院態度：重組還是清算？
- [ ] 催化劑時間表：破產程序進度？
- [ ] 流動性：能否順利退出？

**風險控制**
- 分散持有多個困境債（降低單一風險）
- 避免槓桿（流動性可能消失）
- 深入研究破產法律流程

**案例參考**：2020 年能源公司破產潮、2008 年金融機構困境債

---

### 🎯 策略 3：分拆套利（Spin-off）

**適用場景**：母公司分拆子公司，市場忽視價值

**Lynch + Klarman 框架**
- [ ] 為何分拆？業務不相關？解鎖價值？
- [ ] 子公司獨立後的競爭力？
- [ ] 管理層激勵：有股權嗎？
- [ ] 被動拋售壓力：指數基金必須賣出？
- [ ] Sum-of-parts 估值 vs 市價
- [ ] 分拆後債務分配合理嗎？

**操作時機**
- 分拆後 3-6 個月（被動拋售結束）
- 子公司首次財報發布後（信息更清晰）

**案例參考**：eBay 分拆 PayPal（2015）、GE 分拆 GE Healthcare（2023）

---

### 🎯 策略 4：股東行動主義（Activist Investing）

**適用場景**：公司有價值但管理層/治理有問題

**Icahn 框架**
- [ ] 資產價值明顯被低估？
- [ ] 可解鎖價值路徑：回購、分拆、出售、換管理層？
- [ ] 你有影響力嗎？持股比例？投票權？
- [ ] 其他股東支持度：機構投資者態度？
- [ ] 成本 vs 回報：法律費用、時間成本？
- [ ] 管理層反應：友善還是敵對？

**操作層級**
1. **被動持股**：等其他行動派股東推動
2. **公開信**：施壓但不入董事會
3. **委託書戰**：爭奪董事會席位
4. **全面收購**：私有化

**案例參考**：Icahn vs Apple（2013-2016）、Third Point vs Yahoo

---

### 🎯 策略 5：波動率交易（Volatility Trading）

**適用場景**：隱含波動率 (IV) 與歷史波動率 (HV) 或預期不符

**Burry + 期權策略**
- [ ] IV vs HV：高估（賣出 IV）還是低估（買入 IV）？
- [ ] 事件驅動：財報前 IV 飆升，事件後 IV 崩潰？
- [ ] VIX 指數：當前絕對水平？歷史分位數？
- [ ] 期權偏斜：看跌期權溢價（尾部保護需求）？
- [ ] 時間衰減 (Theta) vs 方向性 (Delta)

**策略組合**
| 策略 | 看法 | 操作 |
|------|------|------|
| 賣出備兌看漲 (Covered Call) | IV 高估 | 持股 + 賣出 Call |
| 保護性看跌 (Protective Put) | 尾部風險 | 持股 + 買入 Put |
| 跨式套利 (Straddle) | 預期大波動 | 買入 Call + Put |
| 勒式套利 (Strangle) | 預期大波動（便宜版） | 買入虛值 Call + Put |
| 鐵鷹式 (Iron Condor) | 預期低波動 | 賣出區間，收取溢價 |

**案例參考**：2020.03 VIX 飆升至 80+，2017 VIX 長期低於 12

---

### 🎯 策略 6：宏觀對沖（Macro Hedge）

**適用場景**：發現政策失衡或大週期轉折

**Soros + Druckenmiller 框架**
- [ ] 失衡點：匯率釘住不可持續？債務爆炸？通膨失控？
- [ ] 反身性：市場參與者行為會加劇失衡？
- [ ] 政策空間：央行還有彈藥嗎？
- [ ] 時機判斷：催化劑是什麼？
- [ ] 倉位管理：確信度高 → 重倉；確信度低 → 輕試探

**經典宏觀交易**
| 目標 | 工具 | 案例 |
|------|------|------|
| 匯率失衡 | 外匯遠期/期貨 | Soros 狙擊英鎊（1992） |
| 利率轉向 | 國債期貨 | Druckenmiller 2000 年重倉國債 |
| 通膨/通縮 | TIPS vs 國債 | 2008 通縮、2021 通膨 |
| 商品周期 | 石油/金屬期貨 | 2020 原油負價格 |
| 信用周期 | CDS、高收益債 | 2008 金融危機做空次貸 |

**案例參考**：1992 英鎊危機、1997 亞洲金融風暴、2008 次貸危機

---

### 🎯 策略 7：統計套利（Statistical Arbitrage）

**適用場景**：歷史相關性偏離，預期均值回歸

**Simons 框架（簡化版）**
- [ ] 配對選擇：同行業、相關性 > 0.7？
- [ ] 偏離程度：Z-score > 2？
- [ ] 均值回歸機制：套利力量存在嗎？
- [ ] 風險控制：最大虧損設定？
- [ ] 交易成本：手續費+滑點會吃掉利潤嗎？
- [ ] 持有期：預期多久回歸？

**配對交易示例**
```
可口可樂 vs 百事可樂
JPM vs BAC
XLE (能源 ETF) vs USO (原油 ETF)
```

**操作**
- 做多相對便宜的
- 做空相對貴的
- 等待價差收斂

**注意事項**
- ⚠️ 相關性可能徹底改變（如行業顛覆）
- ⚠️ 流動性消失時無法平倉
- ⚠️ 槓桿放大風險

**案例參考**：2020.03 市場恐慌時所有相關性崩潰

---

### 🎯 策略 8：可轉債套利（Convertible Arbitrage）

**適用場景**：可轉債定價錯誤（債券價值 + 轉換權價值）

**Marks + 期權理論**
- [ ] 轉換溢價：當前價格 vs 轉股價值？
- [ ] 債券價值（下檔保護）：到期收益率？信用評級？
- [ ] 轉換權價值：隱含波動率 vs 歷史波動率？
- [ ] Delta 對沖：需要做空多少正股？
- [ ] 信用利差：如果股價暴跌，債券價值撐得住嗎？

**基本策略**
```
買入可轉債
做空 Delta 份額的正股
動態調整對沖比例
賺取：票息 + 波動率套利 + 信用利差
```

**風險**
- Gamma 風險（股價大幅波動）
- 流動性風險（可轉債市場較小）
- 信用風險（發行人違約）

---

### 🎯 策略 9：尾部對沖（Tail Hedging）

**適用場景**：保護組合免受黑天鵝事件衝擊

**Marks + Taleb 理念**
- [ ] 保護成本：虛值看跌期權成本？VIX 產品費用？
- [ ] 對沖比例：保護 50%？100%？
- [ ] 觸發條件：跌幅達到多少啟動？
- [ ] 持有期：持續滾動還是事件前購買？
- [ ] 成本拖累 vs 保護效果：年化成本 vs 預期損失

**工具選擇**
| 工具 | 成本 | 效果 | 適用 |
|------|------|------|------|
| 虛值看跌期權 | 中 | 精確保護 | 單一資產 |
| VIX 看漲期權 | 高 | 波動率暴增受益 | 系統性風險 |
| 尾部風險基金 | 費用 | 專業管理 | 大資金 |
| 現金/國債 | 機會成本 | 最簡單 | 所有人 |

**案例參考**：Universa Investments（Taleb）在 2020.03 大賺

---

### 🎯 策略 10：不良資產收購（Non-Performing Assets）

**適用場景**：銀行/基金被迫拋售，價格遠低於真實價值

**Klarman + Marks 框架**
- [ ] 賣方壓力來源：監管要求？流動性危機？
- [ ] 真實價值：底層資產（房地產、貸款）值多少？
- [ ] 回收週期：多久能實現價值？
- [ ] 專業能力：你能管理/處置這些資產嗎？
- [ ] 法律風險：產權清晰嗎？
- [ ] 批量折扣 vs 逐項估值

**經典案例**
- 2008-2012 美國房地產不良資產
- 歐債危機後的南歐銀行 NPL
- 中國 90 年代四大 AMC

## 三、情境劇本

### 劇本 A：市場恐慌時（如 2008、2020.03）

| 步驟 | 參考 | 行動 |
|------|------|------|
| 1 | Marks | 確認週期位置：恐慌是否到極端？ |
| 2 | Klarman | 檢查現金儲備：有彈藥嗎？ |
| 3 | Buffett | 找優質公司被錯殺的機會 |
| 4 | Druckenmiller | 觀察流動性：央行是否轉向？ |
| 5 | Soros | 判斷反身性：恐慌是否在自我強化？ |

**關鍵問題**：
- ✅ 這是流動性危機還是償付危機？
- ✅ 央行有救市意願和能力嗎？
- ✅ 哪些資產被「無差別拋售」？

---

### 劇本 B：市場狂熱時（如 1999、2021）

| 步驟 | 參考 | 行動 |
|------|------|------|
| 1 | Marks | 確認週期位置：貪婪到什麼程度？ |
| 2 | Munger | 檢查自己：有沒有 FOMO？ |
| 3 | Burry | 尋找做空機會（謹慎） |
| 4 | Klarman | 提高現金比例，不追高 |
| 5 | Soros | 判斷泡沫階段：還在自我強化？ |

**關鍵問題**：
- ✅ 估值用什麼邏輯都合理化了嗎？
- ✅ 新手大量進場了嗎？
- ✅ 槓桿水平如何？

---

### 劇本 C：經濟衰退確認時

| 步驟 | 參考 | 行動 |
|------|------|------|
| 1 | Dalio | 確認象限：增長↓ + 通膨？ |
| 2 | Marks | 信用週期：違約開始了嗎？ |
| 3 | Klarman | 關注困境債/特殊情況 |
| 4 | Buffett | 防守：持有現金流強的公司 |
| 5 | Druckenmiller | 觀察政策：何時轉鴿？ |

**關鍵問題**：
- ✅ 衰退深度預期？
- ✅ 央行政策空間？
- ✅ 哪些行業先復甦？

---

### 劇本 D：利率轉向時（加息→降息）

| 步驟 | 參考 | 行動 |
|------|------|------|
| 1 | Druckenmiller | 確認 Fed 轉向信號 |
| 2 | Dalio | 調整資產配置（增久期） |
| 3 | Lynch | 尋找受益於低利率的成長股 |
| 4 | Marks | 信用利差是否開始收窄？ |

**關鍵問題**：
- ✅ 是預防性降息還是衰退式降息？
- ✅ 市場定價了多少降息？
- ✅ 哪些資產對利率最敏感？

## 四、綜合檢查清單

### 📋 買入前檢查（逐項過）

**基本面（Buffett/Lynch）**
- [ ] 商業模式清晰？能用 2 句話解釋？
- [ ] 護城河：品牌/網絡/成本/轉換成本？
- [ ] 管理層：誠信？資本配置能力？
- [ ] 財務：現金流為正？負債合理？

**估值（Buffett/Lynch/Klarman）**
- [ ] 內在價值估算（保守假設）？
- [ ] 安全邊際 > 30%？
- [ ] PEG（成長股）< 1？
- [ ] 為何便宜？賣壓來源？

**宏觀（Dalio/Druckenmiller）**
- [ ] 當前經濟象限？對標的有利？
- [ ] 流動性趨勢？
- [ ] 利率/通膨對估值的影響？

**決策品質（Munger）**
- [ ] 有確認偏誤嗎？
- [ ] 論點能被證偽嗎？
- [ ] 最壞情況是什麼？能接受嗎？

**風控（Marks/Soros）**
- [ ] 下檔風險？永久虧損可能？
- [ ] 止損點設好了嗎？
- [ ] 倉位與確信度匹配？

---

### 📋 持倉定期檢查（每月/季）

- [ ] 買入論點還成立嗎？
- [ ] 基本面有變化嗎？
- [ ] 估值還合理嗎？
- [ ] 宏觀環境變了嗎？
- [ ] 有更好的機會嗎？
- [ ] 該加碼、持有、還是減碼？

---

### 📋 賣出信號檢查

**Buffett 視角**
- [ ] 護城河被侵蝕了？
- [ ] 管理層變差了？
- [ ] 估值太貴了？

**Lynch 視角**
- [ ] 成長故事結束了？
- [ ] PEG > 2？

**Marks 視角**
- [ ] 週期到頂了？
- [ ] 市場太樂觀了？

**Soros 視角**

## 五、DECISION_RULES（IF-THEN 決策規則）

### 5.1 選股決策規則

#### 護城河評估（Buffett/Munger）
```
IF 品牌/網絡效應/成本優勢/轉換成本 存在
   AND ROIC > WACC 持續 5 年+
   THEN 護城河可能存在
   BECAUSE 持續的超額回報是護城河證據

IF 競爭者能輕易複製 OR 技術快速迭代
   THEN 無護城河
   BECAUSE 無持久競爭優勢

IF 護城河依賴單一因素（專利到期、單一客戶）
   THEN 謹慎評估
   BECAUSE 可能是暫時性護城河
```

#### 估值判斷（Buffett/Lynch/Klarman）
```
IF 盈餘收益率 > 10Y國債 × 1.5 AND 護城河完好
   THEN 可能低估
   BECAUSE 安全邊際足夠

IF PEG < 1 AND 增速可持續 5 年+
   THEN 成長股機會
   BECAUSE 成長被低估

IF PEG > 2 OR 盈餘收益率 < 國債收益率
   THEN 高估風險
   BECAUSE 為增長付太多溢價

IF 安全邊際 < 30%
   THEN 不買
   BECAUSE 保護不足
```

#### 價值陷阱識別（Munger）
```
IF 便宜因為行業結構性衰退
   THEN 價值陷阱
   BECAUSE 趨勢不可逆

IF 便宜因為護城河被侵蝕 AND 無改善跡象
   THEN 價值陷阱
   BECAUSE 便宜會更便宜

IF 便宜因為被動拋售（指數調整、評級下調）AND 基本面未變
   THEN 機會
   BECAUSE 非理性賣壓
```

### 5.2 宏觀/擇時決策規則

#### 經濟週期定位（Dalio）
```
IF 增長↑ + 通膨↓ (溫和擴張)
   THEN 偏重股票、公司債
   BECAUSE 最佳環境

IF 增長↑ + 通膨↑ (過熱)
   THEN 偏重商品、TIPS、新興股票
   BECAUSE 通膨受益

IF 增長↓ + 通膨↑ (滯脹)
   THEN 偏重商品、黃金、TIPS
   BECAUSE 最難環境，需硬資產保護

IF 增長↓ + 通膨↓ (衰退/通縮)
   THEN 偏重長期國債、現金
   BECAUSE 避險資產獲益
```

#### 流動性判斷（Druckenmiller）
```
IF 淨流動性（Fed B/S − TGA − RRP）上升
   THEN 做多風險資產
   BECAUSE 流動性推動市場

IF 淨流動性下降 AND 高估值
   THEN 減倉或做空
   BECAUSE 雙殺風險

IF Fed 轉向信號確認（緊→鬆）
   THEN 先佈局成長股
   BECAUSE 流動性改善先反映在股市

IF 最後一次加息確認
   THEN 做多長期國債
   BECAUSE 歷史規律
```

#### 週期判斷（Marks）
```
IF 恐慌拋售 + 估值極低 + 信用凍結
   THEN 週期底部接近
   BECAUSE 極端後必回歸

IF 槓桿激增 + 散戶湧入 + 低質資產熱捧
   THEN 週期頂部接近
   BECAUSE 週期過熱

IF 「這次不一樣」成為共識
   THEN 提高警惕
   BECAUSE 週期規律不變
```

### 5.3 決策品質檢查規則（Munger）

#### 心理偏誤檢查
```
IF 只看到利好信息
   THEN 暫停，主動找反駁論點
   BECAUSE 確認偏誤

IF 「大家都在買」
   THEN 暫停，獨立分析
   BECAUSE 從眾效應

IF 「我比市場聰明」
   THEN 謙虛，問「市場知道什麼我不知道的」
   BECAUSE 過度自信

IF 被成本價錨定
   THEN 忘記成本，重新評估
   BECAUSE 錨定效應

IF 虧損不想賣，盈利急著賣
   THEN 論點對不對？與成本無關
   BECAUSE 損失厭惡

IF 已經說了會買，不想反悔
   THEN 新信息改變判斷就改
   BECAUSE 承諾一致性
```

#### 能力圈檢查
```
IF 無法用 2 句話解釋商業模式
   THEN 不買
   BECAUSE 能力圈外

IF 需要「信仰」未來才能合理化估值
   THEN 不買
   BECAUSE 投機而非投資

IF 論點需要多重假設才能成立
   THEN 降低確信度
   BECAUSE 還沒想清楚
```

### 5.4 倉位與風控規則

#### 倉位管理（Druckenmiller）
```
IF 確信度 90%+ AND 深度研究 AND 邏輯一句話講清楚
   THEN 重倉 10-25%
   BECAUSE 看對方向要敢下注

IF 確信度 60-80%
   THEN 中等倉位 5-10%
   BECAUSE 有空間調整

IF 確信度 < 50%
   THEN 小倉位或不做
   BECAUSE 還沒想清楚

IF 方向做多 BUT 市場寬度惡化
   THEN 警惕，考慮減倉
   BECAUSE 市場內部轉弱
```

#### 止損規則（Soros）
```
IF 買入論點不再成立
   THEN 立即止損
   BECAUSE 論點錯了就跑

IF 基本面惡化（非僅價格下跌）
   THEN 賣出
   BECAUSE 情況在變壞

IF 虧損超過預設止損點（如 -20%）
   THEN 強制執行止損
   BECAUSE 紀律優先

IF 有更好的機會
   THEN 考慮換倉
   BECAUSE 機會成本
```

#### 現金管理（Klarman/Marks）
```
IF 估值普遍高估 AND 缺乏安全邊際
   THEN 提高現金比例（20-50%）
   BECAUSE 等待機會的期權

IF 恐慌拋售 AND 優質資產便宜
   THEN 減少現金，部署
   BECAUSE 機會窗口

IF 不確定
   THEN 保持 20-30% 現金
   BECAUSE 保留彈性
```

### 5.5 賣出決策規則

#### Buffett 視角
```
IF 護城河被侵蝕 AND 無改善跡象
   THEN 賣出
   BECAUSE 競爭優勢消失

IF 估值 > 內在價值 × 1.5
   THEN 考慮減倉
   BECAUSE 安全邊際消失

IF 管理層誠信問題
   THEN 立即賣出
   BECAUSE 無法信任
```

#### Lynch 視角
```
IF 成長故事結束（增速持續下滑）
   THEN 賣出
   BECAUSE 成長股失去成長

IF PEG > 2
   THEN 考慮賣出
   BECAUSE 為成長付太多
```

#### Marks 視角
```
IF 週期到頂 AND 市場過度樂觀
   THEN 減碼
   BECAUSE 逆向操作

IF 信用週期轉向（利差開始擴大）
   THEN 減少信用暴露
   BECAUSE 風險上升
```

### 5.6 特殊情境決策規則

#### 市場恐慌時
```
IF 恐慌拋售 + 估值極低 + 有現金儲備
   THEN 準備買入優質資產
   BECAUSE 別人恐懼時貪婪

IF 是流動性危機 AND 央行有救市能力
   THEN 等待政策底，然後買入
   BECAUSE 流動性危機可解決

IF 是償付危機
   THEN 謹慎，等待明確信號
   BECAUSE 可能更深
```

#### 市場狂熱時
```
IF 槓桿激增 + 散戶湧入 + 「這次不一樣」
   THEN 減倉，提高現金
   BECAUSE 週期頂部特徵

IF 估值需要新指標合理化（用戶數代替盈利）
   THEN 不參與
   BECAUSE 泡沫信號

IF IPO 狂熱，虧損公司上市首日翻倍
   THEN 保持距離
   BECAUSE 經典頂部信號
```

#### 利率轉向時
```
IF 加息結束信號確認
   THEN 做多長期國債
   BECAUSE 歷史規律

IF 是預防性降息（非衰退）
   THEN 做多成長股
   BECAUSE 估值擴張

IF 是衰退式降息
   THEN 防守優先
   BECAUSE 盈利會下滑
```
