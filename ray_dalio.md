# Ray Dalio 投资框架

## QUICK_FACTS
- **基金**：Bridgewater Associates

> 重点：原则化决策、全天候配置、债务周期、极端分散；「经济机器」思维。

## 核心原则
1. **原则驱动**：将决策系统化为可重复规则；减少情绪干扰。
2. **全天候配置（All Weather）**：依据经济环境（增长↑↓ × 通膨↑↓）分散风险，而非仅靠股债比。
3. **债务周期视角**：短期债务周期（5-8 年）+ 长期债务周期（75-100 年）决定宏观环境。
4. **极端分散**：持有 15-20 个低相关资产可大幅降低风险而不损失期望收益。
5. **风险平价**：按波动率而非市值配置；每类资产贡献相近风险。

## All Weather 四象限
| 环境 | 增长 | 通膨 | 偏好资产 |
|------|------|------|----------|
| 温和扩张 | ↑ | ↓ | 股票、公司债 |
| 过热 | ↑ | ↑ | 商品、TIPS、新兴股票 |
| 滞胀 | ↓ | ↑ | 商品、黄金、TIPS |
| 衰退/通缩 | ↓ | ↓ | 长期国债、现金 |

## 债务周期框架
| 阶段 | 特征 | 行动 |
|------|------|------|
| 早期扩张 | 低杠杆、信用易得、利率低 | 做多风险资产 |
| 泡沫 | 杠杆激增、资产价格脱离基本面 | 减曝险、增现金/黄金 |
| 去杠杆 | 违约攀升、信用收缩、央行介入 | 防守；等待政策底 |
| 再通胀 | 央行大规模刺激、货币贬值 | 做多硬资产、股票 |

## 快速检查清单
- **当前象限？** 增长 & 通膨趋势判断 → 调整资产权重。
- **债务周期位置？** 杠杆水平、信用利差、央行政策方向。
- **组合相关性？** 资产间相关性是否在极端时上升？
- **风险预算？** 各资产贡献的组合波动是否均衡？
- **压力测试？** 1930s、2008、2020 等极端情境下回撤如何？

## 与宏观/流动性的连结
- **利率周期**：降息环境有利长期债券；加息环境减少久期。
- **QE/QT**：QE → 资产价格膨胀；QT → 流动性收紧，风险资产承压。
- **美元周期**：强美元 → 减少新兴、商品；弱美元 → 增加。
- **通膨预期**：上升 → 增 TIPS、商品、黄金；下降 → 增长期国债。

## DECISION_RULES（决策规则）

### 资产配置规则
```
IF 增長↑ + 通膨↓ (溫和擴張)
   THEN 偏重股票、公司債
   BECAUSE 最佳環境

IF 增長↑ + 通膨↑ (過熱)
   THEN 偏重商品、TIPS、新興股票
   BECAUSE 通膨受益資產

IF 增長↓ + 通膨↑ (滯脹)
   THEN 偏重商品、黃金、TIPS
   BECAUSE 最難環境，需要實體資產保護

IF 增長↓ + 通膨↓ (衰退/通縮)
   THEN 偏重長期國債、現金
   BECAUSE 避險資產、久期獲益
```

### 债务周期规则
```
IF 早期擴張（低槓桿、信用易得）
   THEN 做多風險資產
   BECAUSE 週期上升期

IF 泡沫階段（槓桿激增、估值脫離基本面）
   THEN 減曝險、增現金/黃金
   BECAUSE 週期見頂

IF 去槓桿（違約攀升、信用收縮）
   THEN 防守，等待政策底
   BECAUSE 週期下行

IF 再通脹（央行大規模刺激）
   THEN 做多硬資產、股票
   BECAUSE 政策轉向利好風險資產
```

### 风险平价规则
```
IF 各資產貢獻風險不均衡
   THEN 再平衡
   BECAUSE 維持分散效果

IF 相關性在極端時上升
   THEN 增加現金、減少暴露
   BECAUSE 分散失效

IF 壓力測試顯示回撤 > 容忍度
   THEN 減少風險暴露
   BECAUSE 生存優先
```
