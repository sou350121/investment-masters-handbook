# Ray Dalio 投資框架

## QUICK_FACTS
- **基金**：Bridgewater Associates
- **規模**：$120B
- **核心風格**：經濟四象限 + 風險平價 + 債務週期
- **最佳適用**：資產配置、週期定位、宏觀判斷

> 重點：原則化決策、全天候配置、債務週期、極端分散；「經濟機器」思維。

## 核心原則
1. **原則驅動**：將決策系統化為可重複規則；減少情緒干擾。
2. **全天候配置（All Weather）**：依據經濟環境（增長↑↓ × 通膨↑↓）分散風險，而非僅靠股債比。
3. **債務週期視角**：短期債務週期（5-8 年）+ 長期債務週期（75-100 年）決定宏觀環境。
4. **極端分散**：持有 15-20 個低相關資產可大幅降低風險而不損失期望收益。
5. **風險平價**：按波動率而非市值配置；每類資產貢獻相近風險。

## All Weather 四象限
| 環境 | 增長 | 通膨 | 偏好資產 |
|------|------|------|----------|
| 溫和擴張 | ↑ | ↓ | 股票、公司債 |
| 過熱 | ↑ | ↑ | 商品、TIPS、新興股票 |
| 滯脹 | ↓ | ↑ | 商品、黃金、TIPS |
| 衰退/通縮 | ↓ | ↓ | 長期國債、現金 |

## 債務週期框架
| 階段 | 特徵 | 行動 |
|------|------|------|
| 早期擴張 | 低槓桿、信用易得、利率低 | 做多風險資產 |
| 泡沫 | 槓桿激增、資產價格脫離基本面 | 減曝險、增現金/黃金 |
| 去槓桿 | 違約攀升、信用收縮、央行介入 | 防守；等待政策底 |
| 再通脹 | 央行大規模刺激、貨幣貶值 | 做多硬資產、股票 |

## 快速檢查清單
- **當前象限？** 增長 & 通膨趨勢判斷 → 調整資產權重。
- **債務週期位置？** 槓桿水平、信用利差、央行政策方向。
- **組合相關性？** 資產間相關性是否在極端時上升？
- **風險預算？** 各資產貢獻的組合波動是否均衡？
- **壓力測試？** 1930s、2008、2020 等極端情境下回撤如何？

## 與宏觀/流動性的連結
- **利率週期**：降息環境有利長期債券；加息環境減少久期。
- **QE/QT**：QE → 資產價格膨脹；QT → 流動性收緊，風險資產承壓。
- **美元週期**：強美元 → 減少新興、商品；弱美元 → 增加。
- **通膨預期**：上升 → 增 TIPS、商品、黃金；下降 → 增長期國債。

## DECISION_RULES（決策規則）

### 資產配置規則
```
IF 增長↑ + 通膨↓ (溫和擴張)
   THEN 偏重股票、公司債
   BECAUSE 最佳環境

IF 增長↑ + 通膨↑ (過熱)
   THEN 偏重商品、TIPS、新興股票
   BECAUSE 通膨受益資產

IF 增長↓ + 通膨↑ (滯脹)
   THEN 偏重商品、黃金、TIPS
   BECAUSE 最難環境，需要實體資產保護

IF 增長↓ + 通膨↓ (衰退/通縮)
   THEN 偏重長期國債、現金
   BECAUSE 避險資產、久期獲益
```

### 債務週期規則
```
IF 早期擴張（低槓桿、信用易得）
   THEN 做多風險資產
   BECAUSE 週期上升期

IF 泡沫階段（槓桿激增、估值脫離基本面）
   THEN 減曝險、增現金/黃金
   BECAUSE 週期見頂

IF 去槓桿（違約攀升、信用收縮）
   THEN 防守，等待政策底
   BECAUSE 週期下行

IF 再通脹（央行大規模刺激）
   THEN 做多硬資產、股票
   BECAUSE 政策轉向利好風險資產
```

### 風險平價規則
```
IF 各資產貢獻風險不均衡
   THEN 再平衡
   BECAUSE 維持分散效果

IF 相關性在極端時上升
   THEN 增加現金、減少暴露
   BECAUSE 分散失效

IF 壓力測試顯示回撤 > 容忍度
   THEN 減少風險暴露
   BECAUSE 生存優先
```
