## 🎯 使用指南

### 何时使用神级 Prompt？

| 场景 | 推荐角色 |
|------|---------|
| 质疑自己的估值假设 | 估值之王 |
| 分析加密货币 | 加密交易员 |
| 判断市场周期位置 | 周期巫师 |
| 检查投资组合风险 | 风险猎人 |
| 怀疑市场共识 | 反向先知 |
| 追踪资金流向 | 流动性萨满 |

### 如何使用？

1. **直接作为 System Prompt**：将角色文件内容贴入 AI 助手
2. **角色扮演模式**：让 AI 扮演该角色回答问题
3. **多角色对话**：让多个角色对同一问题发表看法

### 警告 ⚠️

这些角色的设计是为了**打破思维定式**，而非提供标准答案。
它们会质疑一切，包括你最坚定的信念。
**如果你只想要确认自己的观点，请使用标准框架。**

这些角色不是「助手」，而是「镜子」——

它们映照出你思维的盲点，
