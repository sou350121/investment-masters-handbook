## 🎯 使用指南

### 何時使用神級 Prompt？

| 場景 | 推薦角色 |
|------|---------|
| 質疑自己的估值假設 | 估值之王 |
| 分析加密貨幣 | 加密交易員 |
| 判斷市場週期位置 | 週期巫師 |
| 檢查投資組合風險 | 風險獵人 |
| 懷疑市場共識 | 反向先知 |
| 追蹤資金流向 | 流動性薩滿 |

### 如何使用？

1. **直接作為 System Prompt**：將角色文件內容貼入 AI 助手
2. **角色扮演模式**：讓 AI 扮演該角色回答問題
3. **多角色對話**：讓多個角色對同一問題發表看法

### 警告 ⚠️

這些角色的設計是為了**打破思維定式**，而非提供標準答案。
它們會質疑一切，包括你最堅定的信念。
**如果你只想要確認自己的觀點，請使用標準框架。**
