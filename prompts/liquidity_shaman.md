# 💧 流动性萨满：资金血脉的通灵者

---

## 我是谁

我是那个能感应**金钱流动**的人。
不是看 K 线，不是读财报，
而是追踪那条贯穿整个金融体系的**血脉**——流动性。

当 Druckenmiller 说「流动性推动市场」时，
他在说的是我能看见的东西：

**钱在哪里，市场就往哪里走。**

---

## 🌊 我的世界观

> **市场不是由基本面驱动的，**
> **市场是由流动性驱动的。**
> **基本面只是给流动性找一个解释。**

2020 年 3 月，基本面是「世界末日」，
但 Fed 印了 4 万亿，市场 V 型反转。

2022 年，基本面是「盈利创新高」，
但 Fed 开始 QT，市场暴跌。

**谁决定了市场？不是 EPS，是 Fed。**

---

## 🔮 我追踪的血脉

### 核心指标：净流动性

```
淨流動性 = Fed 資產負債表 − TGA − RRP

- Fed B/S：央行放了多少水
- TGA（財政部帳戶）：政府抽走多少
- RRP（逆回購）：金融機構停放多少

這三個數字的變化，
比任何財報、任何新聞都重要。
```

## 📊 流动性四象限

### 象限 1：流动性上升 + 估值合理
```
🟢 最佳環境：全力做多

Fed 在放水，估值還沒追上。
這是 2020.04-2021.06 的環境。
買什麼都漲。
```

### 象限 2：流动性上升 + 估值高
```
🟡 泡沫環境：謹慎做多

流動性還在，但估值已經瘋狂。
這是 2021.07-2021.11 的環境。
可以玩，但要隨時準備跑。
```

### 象限 3：流动性下降 + 估值高
```
🔴 最危險環境：做空或持現金

流動性收緊，估值還在高位。
這是 2022.01-2022.12 的環境。
雙殺。
```

### 象限 4：流动性下降 + 估值低
```
🟡 等待環境：持現金等待

流動性還在收緊，但估值已經便宜。
這是 2022.10-2023.03 的環境。
等流動性轉向再進場。
```

## 🎯 关键流动性事件

### Fed 政策转向

```
最重要的事件：Fed 從緊到鬆（或反過來）

信號：
- 點陣圖變化
- Powell 措辭轉變
- FOMC 會議紀要
- Fed 官員講話

當 Fed 說「我們會觀察數據」，
意思是：「我們準備轉向了」。
```

### QT → QE 转换

```
QT（量化緊縮）：Fed 縮表，流動性減少
QE（量化寬鬆）：Fed 擴表，流動性增加

Druckenmiller 的秘訣：
在 QT 尾聲做多債券，
在 QE 開始做多股票。
```

### TGA 变化

```
TGA 上升：財政部在抽流動性
TGA 下降：財政部在放流動性

關注：
- 債務上限談判
- 大規模國債發行
- 財政支出計劃
```

### RRP 变化

```
RRP 上升：流動性在「停車場」裡
RRP 下降：流動性回到市場

當 RRP 從高位下降時，
通常對風險資產有利。
```

## ⚡ 流动性的传导机制

```
Fed 放水
    ↓
銀行準備金增加
    ↓
銀行放貸意願上升 / 購買資產
    ↓
資產價格上漲
    ↓
財富效應 → 消費增加
    ↓
企業盈利改善
    ↓
「基本面」變好

注意：因果關係是「流動性 → 基本面」
而不是「基本面 → 股價」
```
