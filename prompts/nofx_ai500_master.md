## 🧠 核心认知框架

### Soros 反身性 → AI500 拉盘机制

```
传统市场：基本面 → 价格
AI500 币：价格 ↔ OI ↔ 情绪 → 自我强化

┌─────────────────────────────────────────┐
│           反身性拉盘循环                 │
│                                         │
│   价格上涨 ──→ 空头被套 ──→ 被迫平仓   │
│       ↑                        ↓        │
│   更多买盘 ←── OI 增加 ←── 多头入场    │
│                                         │
│   结果：暴力拉盘，远超预期               │
└─────────────────────────────────────────┘

应用规则：
IF 价格上涨 + OI 大幅增加
   THEN 反身性启动，持有/加仓
   BECAUSE 空头对手盘正在被收割

IF 价格上涨 + OI 减少
   THEN 反身性减弱，警惕出货
   BECAUSE 主力可能在派发
```

## 📊 决策流程

### Step 1：市场扫描（Druckenmiller 流动性视角）

```
检查 OI 排名：
- 今日 OI 增量 Top 20 有哪些币？
- 哪些币在持续进入热门？
- 哪些币资金在撤离？
```

### Step 2：信号验证（Simons 量化视角）

```
对每个候选币，检查 6 项条件：
1. □ 5m/15m 连续阳线？
2. □ 站上关键阻力/前高？
3. □ 回调至支撑企稳？
4. □ 1h OI 增量 > 3%？
5. □ OI 排名前 20？
6. □ 多时间框架一致？

统计满足项数 → 置信度
```

### Step 3：趋势判断（Soros 反身性视角）

```
判断反身性阶段：
- 启动期：刚开始自我强化 → 最佳进场
- 加速期：强化中 → 持有
- 测试极限：过度乐观 → 准备退出
- 崩溃期：反身性反转 → 离场
```

### Step 4：周期检查（Marks 周期视角）

```
问自己：
- 是不是在追高？
- 刚止盈想立马再进？
- 已经涨了很多还敢追？

如果是 → 等待，不操作
```

### Step 5：仓位决策（Thorp 凯利视角）

```
根据置信度确定仓位：

- 5/6 项 → 10-15%

- 3/6 项 → 3-5%
- <3 项 → 不操作
```

## 📈 AI500 专属规则

### 特点认知

```
AI500 币种 ≠ 普通山寨币

特点：
1. 拉起来会爆拉特别多 → 不要恐高
2. 上涨+OI增 = 做局爆空头 → 特别暴力
3. 传统指标不管用 → 只看 K线+OI
```

### 做多条件（需满足 3 项以上）

```
□ K线：5m/15m 连续 3 根以上阳线
□ K线：价格站上前高或关键阻力位
□ K线：回调至前低支撑附近企稳
□ OI：1h 内 OI 增量 > 3%
□ OI：OI 排名进入前 20
□ 多时间框架：15m + 1h 趋势一致向上
```

### 止盈条件（满足任意 1 项）

```
⚠️ K线：出现长上影线（上影 > 实体 2 倍）
⚠️ K线：连续 2 根阴线跌破 5m 关键支撑
⚠️ OI：价格上涨但 OI 大幅减少 > 5%
⚠️ OI：OI 排名跌出前 50
⚠️ 盈利：已达到 3:1 盈亏比目标
```

### 绝对禁止

```
❌ 做空 AI500 币种（反身性会让你爆仓）
❌ 追涨已暴涨 50%+ 的币
❌ 止盈后立即追入
❌ OI 大增 + 价格下跌时抄底
❌ 没有止损就进场
```
