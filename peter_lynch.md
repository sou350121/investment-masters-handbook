# Peter Lynch 投资框架

## QUICK_FACTS
- **基金**：Fidelity Magellan Fund
- **规模**：峰值 $14B (1990)
- **核心风格**：买你懂的 + PEG < 1 + 股票分类法
- **最佳适用**：成长股筛选、PEG 估值、Tenbagger 寻找

> 重点：「买你了解的」、成长股选股、PEG 估值；个人投资者可打败机构。

## 核心原则
1. **买你熟悉的**：日常生活是发现 Tenbagger（十倍股）的起点。
2. **研究公司，而非市场**：了解公司做什么、为何赚钱、竞争优势。
3. **PEG 估值**：PE / 盈利成长率 < 1 是合理、<0.5 便宜。
4. **分类先行**：根据成长特征分类（慢速成长、稳定成长、快速成长、周期、困境反转、资产型），策略各异。
5. **持续追踪**：买入后持续关注基本面；故事变了就卖。

## 六类股票分类法
| 类别 | 特征 | 策略 |
|------|------|------|
| 慢速成长 | 成熟大型股，低增长 | 股息导向；不期待高资本增值 |
| 稳定成长 | 10-12% 稳定增长 | 核心持仓；回档买、涨多卖 |
| 快速成长 | 20-25%+ 增长的小/中型股 | 高潜力但高风险；重点追踪增速是否持续 |
| 周期 | 汽车、航空、钢铁 | 低 PE 不一定便宜；逆周期布局 |
| 困境反转 | 亏损或危机中但有机会转好 | 高赔率低胜率；控制仓位 |
| 资产型 | 帐面资产被低估 | 挖掘隐藏价值（土地、品牌、现金） |

## 快速检查清单
- **你能 2 分钟讲清楚这家公司吗？**（不能 → 不买）
- **PEG**：PE / 预期 EPS 成长？< 1？
- **负债**：负债股本比健康？利息保障？
- **现金流**：自由现金流为正？
- **机构持股**：过低可能是机会（未被发现）；过高可能已定价。
- **管理层**：内部人买/卖？

## 与宏观/周期的连结
- **经济扩张期**：快速成长股、周期股表现佳；Lynch 偏好此时寻找 Tenbagger。
- **衰退/恐慌**：困境反转股机会增加；稳定成长股防御。
- **利率上升**：高成长股估值压力；Lynch 会降低 PE 容忍度。
- **流动性充裕**：小型成长股更受追捧；注意泡沫风险。

## DECISION_RULES（决策规则）

### 买入规则
```
IF PEG < 1 AND 增速可持續
   THEN 考慮買入
   BECAUSE 成長被合理定價

IF PEG < 0.5 AND 增速穩定
   THEN 深入研究
   BECAUSE 可能被低估

IF 能 2 分鐘講清楚公司做什麼 AND 為何賺錢
   THEN 在能力圈內
   BECAUSE 理解是投資基礎

IF 機構持股低 AND 基本面好
   THEN 機會（未被發現）
   BECAUSE 小型股優勢

IF 內部人在買入
   THEN 正面信號
   BECAUSE 管理層看好
```

### 分类策略规则
```
IF 慢速成長股
   THEN 看股息，不期待資本增值
   BECAUSE 成長有限

IF 穩定成長股
   THEN 核心持倉，回檔買漲多賣
   BECAUSE 穩定複利

IF 快速成長股
   THEN 重點追蹤增速是否持續
   BECAUSE 高潛力高風險

IF 週期股
   THEN 低 PE 不一定便宜，逆週期佈局
   BECAUSE 週期規律

IF 困境反轉股
   THEN 控制倉位，高賠率低勝率
   BECAUSE 不確定性大

IF 資產型
   THEN 挖掘隱藏價值（土地、品牌、現金）
   BECAUSE 市場忽視
```

### 卖出规则
```
IF PEG > 2
   THEN 考慮賣出
   BECAUSE 為成長付太多

IF 故事變了（成長停滯、競爭加劇）
   THEN 賣出
   BECAUSE 買入論點不成立

IF 增速持續下滑
   THEN 重新評估
   BECAUSE 成長股失去成長
```

### 不买规则
```
IF 無法 2 分鐘講清楚公司
   THEN 不買
   BECAUSE 不理解

IF PEG > 2
   THEN 不買
   BECAUSE 太貴

IF 負債過高 AND 利息保障倍數低
   THEN 不買
   BECAUSE 財務風險
```
