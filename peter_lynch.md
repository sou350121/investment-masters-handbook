# Peter Lynch 投資框架

## QUICK_FACTS
- **基金**：Fidelity Magellan Fund
- **規模**：峰值 $14B (1990)
- **核心風格**：買你懂的 + PEG < 1 + 股票分類法
- **最佳適用**：成長股篩選、PEG 估值、Tenbagger 尋找

> 重點：「買你了解的」、成長股選股、PEG 估值；個人投資者可打敗機構。

## 核心原則
1. **買你熟悉的**：日常生活是發現 Tenbagger（十倍股）的起點。
2. **研究公司，而非市場**：了解公司做什麼、為何賺錢、競爭優勢。
3. **PEG 估值**：PE / 盈利成長率 < 1 是合理、<0.5 便宜。
4. **分類先行**：根據成長特徵分類（慢速成長、穩定成長、快速成長、週期、困境反轉、資產型），策略各異。
5. **持續追蹤**：買入後持續關注基本面；故事變了就賣。

## 六類股票分類法
| 類別 | 特徵 | 策略 |
|------|------|------|
| 慢速成長 | 成熟大型股，低增長 | 股息導向；不期待高資本增值 |
| 穩定成長 | 10-12% 穩定增長 | 核心持倉；回檔買、漲多賣 |
| 快速成長 | 20-25%+ 增長的小/中型股 | 高潛力但高風險；重點追蹤增速是否持續 |
| 週期 | 汽車、航空、鋼鐵 | 低 PE 不一定便宜；逆週期佈局 |
| 困境反轉 | 虧損或危機中但有機會轉好 | 高賠率低勝率；控制倉位 |
| 資產型 | 帳面資產被低估 | 挖掘隱藏價值（土地、品牌、現金） |

## 快速檢查清單
- **你能 2 分鐘講清楚這家公司嗎？**（不能 → 不買）
- **PEG**：PE / 預期 EPS 成長？< 1？
- **負債**：負債股本比健康？利息保障？
- **現金流**：自由現金流為正？
- **機構持股**：過低可能是機會（未被發現）；過高可能已定價。
- **管理層**：內部人買/賣？

## 與宏觀/週期的連結
- **經濟擴張期**：快速成長股、週期股表現佳；Lynch 偏好此時尋找 Tenbagger。
- **衰退/恐慌**：困境反轉股機會增加；穩定成長股防禦。
- **利率上升**：高成長股估值壓力；Lynch 會降低 PE 容忍度。
- **流動性充裕**：小型成長股更受追捧；注意泡沫風險。

## DECISION_RULES（決策規則）

### 買入規則
```
IF PEG < 1 AND 增速可持續
   THEN 考慮買入
   BECAUSE 成長被合理定價

IF PEG < 0.5 AND 增速穩定
   THEN 深入研究
   BECAUSE 可能被低估

IF 能 2 分鐘講清楚公司做什麼 AND 為何賺錢
   THEN 在能力圈內
   BECAUSE 理解是投資基礎

IF 機構持股低 AND 基本面好
   THEN 機會（未被發現）
   BECAUSE 小型股優勢

IF 內部人在買入
   THEN 正面信號
   BECAUSE 管理層看好
```

### 分類策略規則
```
IF 慢速成長股
   THEN 看股息，不期待資本增值
   BECAUSE 成長有限

IF 穩定成長股
   THEN 核心持倉，回檔買漲多賣
   BECAUSE 穩定複利

IF 快速成長股
   THEN 重點追蹤增速是否持續
   BECAUSE 高潛力高風險

IF 週期股
   THEN 低 PE 不一定便宜，逆週期佈局
   BECAUSE 週期規律

IF 困境反轉股
   THEN 控制倉位，高賠率低勝率
   BECAUSE 不確定性大

IF 資產型
   THEN 挖掘隱藏價值（土地、品牌、現金）
   BECAUSE 市場忽視
```

### 賣出規則
```
IF PEG > 2
   THEN 考慮賣出
   BECAUSE 為成長付太多

IF 故事變了（成長停滯、競爭加劇）
   THEN 賣出
   BECAUSE 買入論點不成立

IF 增速持續下滑
   THEN 重新評估
   BECAUSE 成長股失去成長
```

### 不買規則
```
IF 無法 2 分鐘講清楚公司
   THEN 不買
   BECAUSE 不理解

IF PEG > 2
   THEN 不買
   BECAUSE 太貴

IF 負債過高 AND 利息保障倍數低
   THEN 不買
   BECAUSE 財務風險
```
