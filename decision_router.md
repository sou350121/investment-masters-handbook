# 投資決策路由 (Decision Router)

> 根據你的問題類型，快速定位該參考哪位投資大師。

## 一、選股決策

### 1.1 護城河評估
**問題**：這家公司有護城河嗎？競爭優勢能持續嗎？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Warren Buffett | Seth Klarman | 品牌/網絡/成本/轉換成本？ |
| Charlie Munger | | 護城河是真的還是偽護城河？ |

**決策規則**：
- IF 無法識別明確護城河 THEN 不買 BECAUSE 無競爭優勢
- IF 護城河依賴單一因素（如專利到期）THEN 謹慎 BECAUSE 可能是暫時性

---

### 1.2 成長股估值
**問題**：這個成長股貴不貴？值得買嗎？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Peter Lynch | Warren Buffett | PEG < 1？增速可持續？ |

**決策規則**：
- IF PEG < 0.5 AND 增速可持續 THEN 機會 BECAUSE 成長被低估
- IF PEG > 2 THEN 高估 BECAUSE 為成長付太多溢價
- IF 增速依賴單一產品/市場 THEN 謹慎 BECAUSE 成長可能不持續

---

### 1.3 深度價值/冷門
**問題**：這個便宜的股票是機會還是陷阱？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Seth Klarman | Howard Marks | 為何便宜？賣壓是結構性還是暫時性？ |
| Michael Burry | | 清算價值是多少？ |

**決策規則**：
- IF 便宜因為被動拋售（指數調整）AND 基本面未變 THEN 機會 BECAUSE 非理性賣壓
- IF 便宜因為行業結構性衰退 THEN 不買 BECAUSE 價值陷阱
- IF 清算價值 > 市值 AND 有催化劑 THEN 深入研究 BECAUSE 極端安全邊際

---

### 1.4 價值陷阱排除
**問題**：如何避免買到價值陷阱？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Charlie Munger | Warren Buffett | 便宜的原因會持續惡化嗎？ |
| | Seth Klarman | |

**決策規則**：
- IF 護城河被侵蝕 AND 無改善跡象 THEN 價值陷阱 BECAUSE 便宜會更便宜
- IF 管理層問題 AND 無更換可能 THEN 價值陷阱 BECAUSE 無改變催化劑
- IF 行業結構性衰退（如報紙）THEN 價值陷阱 BECAUSE 趨勢不可逆

---

## 二、宏觀/擇時

### 2.1 經濟週期定位
**問題**：現在經濟在週期的哪個位置？該進攻還是防守？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Ray Dalio | Howard Marks | 增長↑↓？通膨↑↓？ |
| | Stanley Druckenmiller | 處於四象限的哪裡？ |

**決策規則**：
- IF 增長↑ + 通膨↓ THEN 做多股票、公司債 BECAUSE 溫和擴張最佳環境
- IF 增長↓ + 通膨↑ THEN 持有商品、黃金、TIPS BECAUSE 滯脹保護
- IF 增長↓ + 通膨↓ THEN 持有長期國債、現金 BECAUSE 衰退/通縮環境

---

### 2.2 流動性趨勢
**問題**：流動性在擴張還是收縮？對市場有什麼影響？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Stanley Druckenmiller | Ray Dalio | Fed B/S、TGA、RRP 方向？ |

**決策規則**：
- IF 淨流動性（Fed B/S − TGA − RRP）上升 THEN 做多風險資產 BECAUSE 流動性推動市場
- IF 淨流動性下降 AND 高估值 THEN 減倉 BECAUSE 雙殺風險
- IF Fed 政策轉向（緊→鬆）THEN 先佈局成長股 BECAUSE 流動性改善先反映在股市

---

### 2.3 貨幣/匯率機會
**問題**：有貨幣交易機會嗎？匯率有失衡嗎？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| George Soros | Stanley Druckenmiller | 政策/匯率是否不可持續？ |

**決策規則**：
- IF 固定匯率制 AND 外儲耗竭 AND 經濟失衡 THEN 貨幣脫鉤機會 BECAUSE Soros 經典交易
- IF 反身性自我強化階段 THEN 順勢交易 BECAUSE 趨勢會超預期
- IF 政策空間耗盡 THEN 等待崩潰 BECAUSE 不可持續必將結束

---

### 2.4 利率轉向
**問題**：Fed 要轉向了嗎？該如何調整？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Stanley Druckenmiller | Ray Dalio | 加息結束/降息開始的信號？ |
| | Howard Marks | |

**決策規則**：
- IF 最後一次加息確認 THEN 做多長期國債 BECAUSE 歷史規律
- IF 預防性降息（非衰退）THEN 做多成長股 BECAUSE 估值擴張
- IF 衰退式降息 THEN 防守優先 BECAUSE 盈利會下滑

---

## 三、風險檢查

### 3.1 心理偏誤檢查
**問題**：我的決策有偏誤嗎？

| 優先參考 | 核心問題 |
|----------|----------|
| Charlie Munger | 有確認偏誤嗎？有FOMO嗎？有過度自信嗎？ |

**決策規則**：
- IF 只看利好信息 THEN 確認偏誤 BECAUSE 需要主動找反駁論點
- IF "大家都在買" THEN 從眾效應 BECAUSE 需要獨立思考
- IF "我比市場聰明" THEN 過度自信 BECAUSE 市場可能知道你不知道的
- IF "已經買了不能認錯" THEN 承諾一致性 BECAUSE 論點錯了就該走

---

### 3.2 週期位置判斷
**問題**：市場情緒在哪個極端？該貪婪還是恐懼？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Howard Marks | Ray Dalio | 恐懼 vs 貪婪？信用週期在哪？ |

**決策規則**：
- IF 恐慌拋售 + 估值極低 + 信用凍結 THEN 底部接近 BECAUSE 極端後必回歸
- IF 槓桿激增 + 散戶湧入 + 低質資產熱捧 THEN 頂部接近 BECAUSE 週期過熱
- IF 「這次不一樣」成為共識 THEN 警惕 BECAUSE 週期規律不變

---

### 3.3 極端風險/泡沫識別
**問題**：這是泡沫嗎？會崩盤嗎？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Michael Burry | Howard Marks | 共識錯在哪？數據被忽略了嗎？ |
| Charlie Munger | George Soros | |

**決策規則**：
- IF 估值需要「信仰」才能合理化 THEN 泡沫風險 BECAUSE 基本面無法支撐
- IF 新指標取代傳統估值（如用戶數代替盈利）THEN 泡沫信號 BECAUSE 歷史重演
- IF 股神已老/價值投資已死成為共識 THEN 泡沫信號 BECAUSE 經典逆向指標

---

### 3.4 止損決策
**問題**：該認錯止損嗎？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| George Soros | Stanley Druckenmiller | 論點還成立嗎？ |

**決策規則**：
- IF 買入論點不再成立 THEN 立即止損 BECAUSE 錯了就跑
- IF 基本面惡化（非價格下跌）THEN 賣出 BECAUSE 情況在變壞
- IF 有更好的機會 THEN 換倉 BECAUSE 機會成本

---

## 四、組合配置

### 4.1 資產配置
**問題**：該如何配置股/債/商品/現金？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Ray Dalio | Howard Marks | 經濟象限？各資產風險貢獻？ |

**決策規則**：
- IF 增長↑ + 通膨↓ THEN 偏重股票 BECAUSE 最佳環境
- IF 增長↓ + 通膨↑ THEN 偏重商品、黃金 BECAUSE 滯脹保護
- IF 不確定 THEN 全天候配置 BECAUSE 分散風險

---

### 4.2 現金比例
**問題**：該持有多少現金？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Seth Klarman | Howard Marks | 機會夠好嗎？還是該等待？ |
| | Warren Buffett | |

**決策規則**：
- IF 估值普遍高估 AND 缺乏安全邊際 THEN 提高現金 BECAUSE 等待更好機會
- IF 恐慌拋售 AND 優質資產便宜 THEN 減少現金 BECAUSE 機會窗口
- IF 不確定 THEN 保持 20-30% 現金 BECAUSE 彈性

---

### 4.3 集中 vs 分散
**問題**：該集中持股還是分散？

| 集中派 | 分散派 | 選擇依據 |
|--------|--------|----------|
| Warren Buffett | Ray Dalio | 確信度高 → 集中 |
| Stanley Druckenmiller | | 確信度低 → 分散 |

**決策規則**：
- IF 確信度極高（能力圈內 + 深度研究）THEN 集中 BECAUSE 分散是無知的保護
- IF 確信度一般 OR 宏觀不確定 THEN 分散 BECAUSE 降低單一風險
- IF 資金規模大 THEN 必須分散 BECAUSE 流動性限制

---

### 4.4 倉位管理
**問題**：確信度和倉位該如何匹配？

| 優先參考 | 次要參考 | 核心問題 |
|----------|----------|----------|
| Stanley Druckenmiller | George Soros | 方向確信度有多高？ |

**決策規則**：
- IF 確信度 90%+ THEN 重倉（10-25%）BECAUSE 看對方向要敢下注
- IF 確信度 60-70% THEN 中等倉位（3-7%）BECAUSE 需要空間調整
- IF 確信度 < 50% THEN 小倉位或不做 BECAUSE 還沒想清楚

## 七、決策工作流（完整流程）

```
1. 宏觀定位 → 參考 Ray Dalio / Stanley Druckenmiller
   └→ 當前經濟象限？流動性趨勢？

2. 週期判斷 → 參考 Howard Marks
   └→ 市場情緒？信用週期？該進攻還是防守？

3. 標的篩選 → 參考 Warren Buffett / Peter Lynch / Seth Klarman
   └→ 護城河？估值？安全邊際？

4. 決策品質檢查 → 參考 Charlie Munger
   └→ 有偏誤嗎？論點夠簡單清晰嗎？

5. 倉位與風控 → 參考 Stanley Druckenmiller / George Soros
   └→ 確信度 vs 倉位？止損點？
```

> **使用提示**：
> - 找到對應問題類型 → 查看優先參考的投資人 → 閱讀其詳細文檔
> - 決策規則採用 IF-THEN 格式，可直接用於判斷
> - 複雜決策建議交叉參考多位投資人觀點
